# Keyboard Operations

> Category: `Panel` | Subcategory: `Menus` | Type: `concept`

The following key combinations may be used when working in the Panel Designer:

## Calling Help

Opens help.

In dialogs/window, help is displayed for the particular dialog/window.

<F1>

| Function | Key Combination |
|---|---|
| Opens help. In dialogs/window, help is displayed for the particular dialog/window. | <F1> |

## Controlling the Cursor in the Panel’s Working Area

Skip to the next control

<Tab>

Skip to the previous control

<Shift>+<Tab>

Skip to the next panel

<Ctrl>+<Tab>

If the focus is in the panel's working area, you can press <Esc> to select the parent element of the currently selected control.

## Activating Menu Commands

Selects the ribbon and activates the selection of the tabs.

| Function | Key Combination |
|---|---|
| Selects the ribbon and activates the selection of the tabs. After that the following functions are available: | <Alt> |
| Opens the tab for the selected character. | <Character> |
| Opens the menu/window for the selected character on th eselected tab. | <Character> |
| Opens the tab to the left or right of the active tab. | <Arrow key right>/<Arrow key left> |
| Moves the selection of menu commands within the opened menu. | <Arrow key up>/<Arrow key down> |
| Executes the command of the selected menu command. | <Enter> |
