# Screen Format

> Category: `Panel` | Subcategory: `Windows` | Type: `concept`

## Quick Access Toolbar

The menu bar is located along the upper edge of the screen. It is used for selecting commands.

## Ribbon

The ribbon below can be used to select important commands rapidly.

## Windows

Below the ribbon, you find the main window with its docked windows.

## Status Bar

The status bar is located along the lower border of the main Panel Designer window.

Control type and assigned symbol are displayed on it.
