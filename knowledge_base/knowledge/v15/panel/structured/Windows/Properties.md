# Properties

> Category: `Panel` | Subcategory: `Windows` | Type: `concept`

You can use the Properties Window to configure selected elements (panel and controls) on the working area.

The setting options for each selected element are displayed in the Properties Window in the form of a table. With a multi-selection of elements, you can change all the shared settings at once.

## Properties Window Toolbar

Settings are categorized (default setting)

Settings are listed in alphabetical order.

All available settings for the selected control are displayed.

Only the relevant settings for the selected control are displayed.

## Color Selection

In the case of the color selection list, a window with the following tabs will appear: User-defined, Web and System.

If you want to set a user-defined color, you can either select one of the colors available via the User-defined tab or you can enter the RGB value for the color directly behind the color display. The values must be separated by a ";" character. An error message will be displayed if the entry is incorrect.

To open the Color Selection dialog from within the User-defined tab, right-click on any of the empty color fields.You can then select the required color by either entering the RGB values or by selecting it directly from a color scale and clicking the [Add color] button to add it to the palette of user-defined colors.You can also use this dialog to specify the chroma, saturation, and intensity of the color.
