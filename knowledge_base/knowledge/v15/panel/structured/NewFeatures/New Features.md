## Version 7.0

The new Panel Designer facilitates the generation of panels with its user-friendly interface.

The division of the main window into the working area, Toolbox, Symbol Explorer and Properties Window makes the process to configure the panel and its controls virtually dialog-free:

You can select a number of elements at once to modify their common settings simultaneously in the Properties Window.

The snaplines which appear in the working area make it easy for you to align the controls.

Symbol can be assigned directly to a panel using drag and drop. Following assignment, the standard control set in the Symbol Explorer Drag & Drop Configuration dialog is displayed.

The symbol can be assigned to a new control at any time using the control's context menu.

Multiple panels can be opened at the same time in the Panel Designer. The open panels are displayed on separate tabs in the working area.
