# xcpDeactivateAll

> Category: `XCP` | Type: `function`

## Syntax

```c
long xcpDeactivateAll(char[] ecuQualifier);
```

## Description

Deactivates auto read of all configured parameters of the active measurement group.

## Parameters

| Name | Description |
|---|---|
| ecuQualifier | Name of the device – configured within the CANoeXCP/CCP Window. |
