# clear

> Category: `AssociativeFields` | Type: `method`

## Syntax

```c
void map[char[]]::clear();
void map[float]::clear();
void map[int64]::clear();
void map[long]::clear();
```

## Description

Removes all elements from the field.

## Return Values

—

## Example

```c
m.clear();
```
