# size

> Category: `AssociativeFields` | Type: `method`

## Syntax

```c
long map[char[]]::size();
long map[float]::size();
long map[int64]::size();
long map[long]::size();
```

## Description

Returns the number of elements in the field.

## Return Values

Number of elements in the field.

## Example

```c
if (m.size() == 0) write("Map is empty!");
```
