# mkExtId

> Category: `CAN` | Type: `function`

## Syntax

```c
dword mkExtId(dword id);
```

## Description

Returns an extended id.

## Parameters

| Name | Description |
|---|---|
| id | Id part of a message. |

## Return Values

Extended identifier

## Example

```c
...
msg.id = mkExtId(this.id);
...
```
