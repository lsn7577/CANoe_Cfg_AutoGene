# getCardTypeEx

> Category: `CAN` | Type: `function`

## Syntax

```c
int getCardTypeEx(int can);
```

## Description

Determines the card type of CAN channel. Is needed e.g. to program the BTR / OCR values.

## Parameters

| Name | Description |
|---|---|
| CAN | Channel number |
