# MessageTimeNS (FR)

> Category: `FlexRay` | Type: `function`

## Syntax

```c
float ns = MessageTimeNS( this );
```

## Description

This function returns the receive time of the frame or slot in nanoseconds.

## Parameters

| Name | Description |
|---|---|
| this | The function can only be used in the context of the following event procedures: on frFrame on frNullFrame on frFrameError on frSlot on frStartCycle this references the corresponding receive object in the event procedure. |

## Return Values

Receive time in nanoseconds since start of message.

## Example

For an example see event procedure on frFrame.
