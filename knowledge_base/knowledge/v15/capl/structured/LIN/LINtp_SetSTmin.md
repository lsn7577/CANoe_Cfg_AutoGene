# LINtp_SetSTmin

> Category: `LIN` | Type: `function`

## Syntax

```c
void LINtp_SetSTmin(dword STmin_ms);
```

## Description

Sets minimum separation time to use.

## Parameters

| Name | Description |
|---|---|
| STmin_ms | [0, 127] Separation time in milliseconds other: 127 ms will be used |

## Return Values

—
