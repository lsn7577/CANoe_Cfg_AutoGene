# linGetOEMWakeupInd

> Category: `LIN` | Type: `function`

## Syntax

```c
long linGetOEMWakeupInd(); // form 1
long linGetOEMWakeupInd(char slaveName[]); // form 2
```

## Description

With this function it's possible to query the wake-up indication bit of a Slave node.

Without parameter, the wake-up indication bit of the calling slave is returned or zero if the caller is not a slave node.

## Parameters

| Name | Description |
|---|---|
| slaveName | Name of the queried Slave node as specified in the database. |
