# linBusIsAwake

> Category: `LIN` | Type: `function`

## Syntax

```c
long linBusIsAwake();
```

## Description

Queries the awake state of the LIN bus.

## Return Values

Returns a value unequal to zero if the bus is awake, otherwise zero.
