# LINtp_SetNAD

> Category: `LIN` | Type: `function`

## Syntax

```c
long LINtp_SetNAD(long NAD);
```

## Description

In a slave node, sets its node address for communication with the master.

## Parameters

| Name | Description |
|---|---|
| NAD | Node address [1, 126] |
