# linResetRespBaudrate

> Category: `LIN` | Type: `function`

## Syntax

```c
long linResetRespBaudrate(long frameId);
```

## Description

Resets a response baud rate for a specified frame to the master baud rate.

## Parameters

| Name | Description |
|---|---|
| frameId | The identifier of the frame for which the response baud rate will be reset. |

## Return Values

On success, a value unequal to zero, otherwise zero.
