# linSetOEMDataInd

> Category: `LIN` | Type: `function`

## Syntax

```c
long linSetOEMDataInd(long active);
```

## Description

Sets/resets the data indication bit for a calling slave node.

## Parameters

| Name | Description |
|---|---|
| active | 0: set 1: reset |

## Return Values

On success, a value unequal to zero, otherwise zero.
