# LINtp_SetFillByteValue

> Category: `LIN` | Type: `function`

## Syntax

```c
void LINtp_SetFillByteValue(long fillByte);
```

## Description

Sets fill byte used to pad frames.

## Parameters

| Name | Description |
|---|---|
| fillByte | This value will be used to fill up the frames [0, 255]. |

## Return Values
