# linSetRespBaudrate

> Category: `LIN` | Type: `function`

## Syntax

```c
long linSetRespBaudrate(long frameId, long baudrate);
```

## Description

Sets a response baud rate for a specified frame. This baud rate will be used for the response, regardless of the master baud rate.

## Parameters

| Name | Description |
|---|---|
| frameId | The identifier of the frame for which the response baud rate will be set. |
| baudrate | The baud rate to be used for the response. Value range: 1000-20000 |

## Return Values

On success, a value unequal to zero, otherwise zero.
