# linWakeupInternal

> Category: `LIN` | Type: `function`

## Syntax

```c
long linWakeupInternal();
```

## Description

Wakes up the LIN bus without sending any wakeup signals.

## Return Values

On success, a value unequal to zero, otherwise zero.
