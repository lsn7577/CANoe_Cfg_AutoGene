# coTfsODClearAllEntries

> Category: `CANopen` | Type: `function`

## Syntax

```c
long coTfsODClearAllEntries( void );
```

## Description

The function deletes all objects of the internal test object list, which is used by coTfsODChk and coTfsODChkNotExist.

## Return Values

Error code

## Example

```c
see example of coTfsODAddEntry
```
