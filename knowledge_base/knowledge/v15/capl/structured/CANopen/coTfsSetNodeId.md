# coTfsSetNodeId

> Category: `CANopen` | Type: `function`

## Syntax

```c
long coTfsSetNodeId( dword nodeID );
```

## Description

The internal node-ID for the simplified test functions is set.

## Parameters

| Name | Description |
|---|---|
| nodeID | 0 < node-ID < 128 |

## Return Values

Error code

## Example

```c
coTfsSetNodeId (2); // set node-ID = 2
```
