# ScopeSinglePolygonBitMask

> Category: `Scope` | Type: `notes`

## Description

This structure defines a bit mask for the eye diagram analysis.

—

| Keyword | Description | Type | Access Limitations |
|---|---|---|---|
| MaskPointCount | The number of bit mask points | qword | Read-only |
| TimeOffset | Mask Point relative position within bit in % | Double[10] | Read-only |
| Voltage | Mask Point in mV | Long[10] | Read-only |
