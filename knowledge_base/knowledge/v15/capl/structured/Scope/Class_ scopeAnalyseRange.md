# Class: scopeAnalyseRange

> Category: `Scope` | Type: `notes`

## Description

This structure defines the analysis range for e.g. a transition time measurement.

—

testWaitScopePerformSignalTransitionTime

| Keyword | Description | Type | Access Limitations |
|---|---|---|---|
| StartField | The start field of the area. See ScopeBitAnalyse.cin | long | Read/Write |
| StartFieldBit | The bit number within the start field (starts with 0) | long | Read/Write |
| EndField | The end field of the area. | long | Read/Write |
| EndFieldBite | The bit number within the end field (starts with 0) | long | Read/Write |
