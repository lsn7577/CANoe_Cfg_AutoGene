# SCC_GetCertChainVerificationErrorCount

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
dword SCC_GetCertChainVerificationErrorCount( )
```

## Description

Returns the number of verification errors from the last certificate chain verification, which is indicated by the callback function SCC_CertChainVerificationInd. Individual errors can be queried with SCC_GetCertChainVerificationError.

—
