# SCC_GetHeaderData

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetHeaderData ( byte HeaderData[] )
```

## Description

Returns the SDP or V2G message header (to the output buffer).

## Parameters

| Name | Description |
|---|---|
| HeaderData | 8 byte output buffer |

## Return Values

Returns V2G / SDP Transport Layer message header.
