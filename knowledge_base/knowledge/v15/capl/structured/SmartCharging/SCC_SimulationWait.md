# SCC_SimulationWait

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_SimulationWait ( )
```

## Description

This function pauses/resumes the SCC simulation. This functionality can be used to freeze the protocol flow in a certain state.
