# SCC_GetEVSEPort

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_GetEVSEPort ( )
```

## Description

Gets the EVSE port of a SECCDiscoveryRes message.

## Return Values

TCP port transmitted from the charge point.
