# SCC_SLAC_GetAttenResults

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_SLAC_GetAttenResults ( float Results[], long& ResultCount )
```

## Description

Queries the measured average attenuation values during the callback SCC_SLACFinishedInd.

## Parameters

| Name | Description |
|---|---|
| Results | Array to which the attenuation results are written. The results are returned sorted, starting with the lowest attenuation. |
| ResultCount | Variable to which the number of results in the array is written. |

## Return Values

—
