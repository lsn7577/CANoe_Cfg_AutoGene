# SCC_TCPConnectInd

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_TCPConnectInd ( dword Port, long IsInitiator )
```

## Description

The callback is called when a TCP connection is opened.

## Parameters

| Name | Description |
|---|---|
| Port | Port number of the TCP connection. |
| IsInitiator | 1 if the node layer initiated the connection (client case). 0 if the node layer received the connection request (server case). |

## Return Values

—
