# SCC_SetBatterySoC

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_SetBatterySOC ( float BatteryState )
```

## Description

Sets the charging state of the battery, either in [Wh] with the first function or in [%] with the second.

## Parameters

| Name | Description |
|---|---|
| BatteryState | Desired charging state in Wh or percent. |

## Return Values

—
