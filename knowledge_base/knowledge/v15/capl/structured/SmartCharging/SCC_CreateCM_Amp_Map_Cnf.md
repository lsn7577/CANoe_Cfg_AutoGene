# SCC_CreateCM_Amp_Map_Cnf

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_CreateCM_Amp_Map_Cnf ( byte SourceMac[], byte TargetMac[], dword ResType )
```

## Description

Creates a CM_Amp_Map.Cnf message for sending.

## Parameters

| Name | Description |
|---|---|
| SourceMac | Source address of the Ethernet frame. |
| TargetMac | Destination address of the Ethernet frame. |
| ResTyp | 0x00 = Success 0x01 = Failure |
