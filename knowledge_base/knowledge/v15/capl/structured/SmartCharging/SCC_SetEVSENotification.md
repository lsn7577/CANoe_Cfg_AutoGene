# SCC_SetEVSENotification

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_SetEVSENotification ( char EVSENotification[] )
```

## Description

Sets the EVSE notification enum.

## Parameters

| Name | Description |
|---|---|
| EVSENotification | String buffer that is written to the EVSE notification. The string should be a valid enum value according to the schema. |
