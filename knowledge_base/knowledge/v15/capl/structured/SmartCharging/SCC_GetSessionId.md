# SCC_GetSessionId

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetSessionId ( byte SessionID [] )
```

## Description

Outputs the SessionID of a connection.

## Parameters

| Name | Description |
|---|---|
| SessionID | 8-byte long array to which the SessionID is written. |
