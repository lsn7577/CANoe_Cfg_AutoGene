# on transmit_any_announce

> Category: `DistributedObjects` | Type: `event`

## Description

This handler is called whenever an instance of a provided data, event, or field member transmits an announce via the CAPL binding.

If the member does not use the CAPL binding, the handler is ignored.

If there are multiple handlers for the same member, the measurement will be aborted.

Inside the handler the this expression refers to the parent object of the member.
