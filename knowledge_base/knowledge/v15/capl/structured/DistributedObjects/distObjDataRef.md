# distObjDataRef

> Category: `DistributedObjects` | Type: `function`

## Syntax

```c
distObjDataRef <Member>
```

## Description

References a distributed object data member. This type cannot be used in declarations. This is a subtype of distObjMemberRef <Member> and valueHandle <DataType>. The referenced value is immutable for consumed members and mutable otherwise.

## Parameters

| Name | Description |
|---|---|
| Member | Path of a distributed object data member. |

## Example

```c
on start
{
  distObjRef ExampleInterface object = ExampleObject;
  object.ExampleDataMember;
}
```
