# Subtype

> Category: `Media` | Type: `notes`

## Description

Subtype

Subtype string for a media type.

string

The subtype string defines a specific media format type within a major type.For example, within video, the subtypes include RGB-24, RGB-32, UYVY, AYUV, and so forth. Within audio, the subtypes include PCM audio, Windows Media Audio 9, and so forth.For possible values, see different Subtypes.
