# ethernetPacket::GetDestinationIPEndpoint

> Category: `IP` | Type: `method`

## Syntax

```c
long ethernetPacket.GetDestinationIPEndpoint( ip_Endpoint endpoint );
```

## Description

Assigns the destination IP address and UDP/TCP port number to an ip_Endpoint variable.

## Parameters

| Name | Description |
|---|---|
| endpoint | IP address and port is assigned to this variable. |
