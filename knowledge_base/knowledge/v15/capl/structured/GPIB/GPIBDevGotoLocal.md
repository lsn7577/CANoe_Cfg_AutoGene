# GPIBDevGotoLocal

> Category: `GPIB` | Type: `function`

## Syntax

```c
long GPIBDevGotoLocal (long deviceDescriptor);
```

## Description

Sets a device into local mode. A successor GPIB command sets the device back into remote mode.

## Parameters

| Name | Description |
|---|---|
| deviceDescriptor | Device ID |
