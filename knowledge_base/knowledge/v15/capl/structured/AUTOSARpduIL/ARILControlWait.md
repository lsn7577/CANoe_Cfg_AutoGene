# ARILControlWait

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILControlWait (dword flags)
```

## Description

Stops sending, but accepts signal changes.

## Parameters

| Name | Description |
|---|---|
| flags | The value defines the desired wait behavior: 0: IL will enter state waiting (see Interaction Layer State Model). The IL can be activated by calling ARILControlResume. 1: IL will enter state suspended (see Interaction Layer State Model). This is the same effect as calling function ARILControlStop. The IL can be activated by calling ARILControlStart. |
