# TCIL_GetState

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long TCIL_GetState(); // form 1
long TCIL_GetState(dbNode tc); // form 2
```

## Description

Returns the current state of the TC IL.

## Parameters

| Name | Description |
|---|---|
| tc | TC simulation node to apply the function |

## Example

Example form 1

```c
on key 's'
{
  long state;
  state = TCIL_GetState();
}
```
