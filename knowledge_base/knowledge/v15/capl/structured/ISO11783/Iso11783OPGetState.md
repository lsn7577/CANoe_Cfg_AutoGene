# Iso11783OPGetState

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long Iso11783OPGetState( dword ecuHandle );
```

## Description

Returns the state of the Object Pool API.

## Parameters

| Name | Description |
|---|---|
| ecuHandle | Handle of the ECU. The handle must be created with Iso11783CreateECU. |

## Example

```c
state = Iso11783OPGetState( handle );
```
