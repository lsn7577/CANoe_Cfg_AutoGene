# Iso11783IL_TIMResetServerStatus

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long Iso11783IL_TIMResetServerStatus( ); // form 1
long Iso11783IL_TIMResetServerStatus(dbNode server); // form 2
```

## Description

This function reverts changes made by Iso11783IL_TIMSetServerStatus and turn back to the default behavior of the TIM server.

## Parameters

| Name | Description |
|---|---|
| server | TIM server node. |
