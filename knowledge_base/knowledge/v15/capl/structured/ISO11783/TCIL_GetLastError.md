# TCIL_GetLastError

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long TCIL_GetLastError(); // form 1
long TCIL_GetLastError(dbNode tc); // form 2
```

## Description

Gets the return value of the last called TC IL function.

## Parameters

| Name | Description |
|---|---|
| tc | TC simulation node to apply the function |
