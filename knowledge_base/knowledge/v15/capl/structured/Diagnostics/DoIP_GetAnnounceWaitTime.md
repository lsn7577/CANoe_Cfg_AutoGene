# DoIP_GetAnnounceWaitTime

> Category: `Diagnostics` | Type: `function`

## Syntax

```c
dword DoIP_GetAnnounceWaitTime();
```

## Description

Gets the delay before the sending of Vehicle Announcement Messages starts in a vehicle simulation (in milliseconds).

## Return Values

The delay before the sending of vehicle announcement messages starts in a vehicle simulation (configured in the DoIP.INI or via DoIP_ConfigureVehicleAnnouncement).
