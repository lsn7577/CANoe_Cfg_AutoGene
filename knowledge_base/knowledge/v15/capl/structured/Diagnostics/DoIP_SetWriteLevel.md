# DoIP_SetWriteLevel

> Category: `Diagnostics` | Type: `function`

## Syntax

```c
void DoIP_SetWriteLevel(long level);
```

## Description

Sets the amount of messages written to the CANoe Write Window by the DoIP implementation.

## Parameters

| Name | Description |
|---|---|
| Level | 0: no output 1: only errors 2: also warnings Higher values are only relevant during development. |

## Return Values

—
