# diagIsPositiveResponse

> Category: `Diagnostics` | Type: `function`

## Syntax

```c
long diagIsPositiveResponse (diagResponse obj)
```

## Description

Returns value <> 0 if the object is a positive response to a request.

## Parameters

| Name | Description |
|---|---|
| obj | Diagnostics object |

## Return Values

0 or <>0
