# diagIsRaw

> Category: `Diagnostics` | Type: `function`

## Syntax

```c
long diagIsRaw( diagRequest request);
long diagIsRaw( diagResponse response);
```

## Description

Returns if the object is stored as raw data or can be interpreted.

## Parameters

| Name | Description |
|---|---|
| request | Request |
| response | Response |
