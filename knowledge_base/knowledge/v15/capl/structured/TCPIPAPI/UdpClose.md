# UdpClose

> Category: `TCPIPAPI` | Type: `function`

## Syntax

```c
long UdpClose( dword socket);
```

## Description

The function closes the UDP socket. Upon successful completion the passed socket is no longer valid.

## Parameters

| Name | Description |
|---|---|
| socket | The socket to be closed. |
