# DeleteMapObject

> Category: `MapWindowAPI` | Type: `function`

## Syntax

```c
long DeleteMapObject( long handle );
```

## Description

Deletes the map object assigned to this handle in the Map Window.

## Parameters

| Name | Description |
|---|---|
| handle | Reference of the map object. |

## Return Values

Number of deleted objects.

See Map Window API example
