# CreateMapObject

> Category: `MapWindowAPI` | Type: `function`

## Syntax

```c
long CreateMapObject( long objectType );
```

## Description

Initializes a map object to be drawn in the Map Window.

## Parameters

| Name | Description |
|---|---|
| objectType | Figure to be drawn in the Map Window: 1: Square 2: Rectangle 3: Ellipse 4: Cross 5: Triangle 6: Line 7: Bitmap 8: Text 9: Vehicle |

## Return Values

Handle of the created map object.
