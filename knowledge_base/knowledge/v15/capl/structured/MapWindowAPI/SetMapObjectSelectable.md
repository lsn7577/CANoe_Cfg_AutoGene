# SetMapObjectSelectable

> Category: `MapWindowAPI` | Type: `function`

## Syntax

```c
long SetMapObjectSelectable( long handle, long selectable );
```

## Description

Defines if a map object will raise the callback OnMapObjectClick when it is clicked with the mouse cursor.

## Parameters

| Name | Description |
|---|---|
| handle | Handle of the map object. |
| Selectable | 1 = true, 0 = false. |
