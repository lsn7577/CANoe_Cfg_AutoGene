# SetMapObjectVisible

> Category: `MapWindowAPI` | Type: `function`

## Syntax

```c
long SetMapObjectVisible ( long handle, int visible );
```

## Description

Sets the visibility of a map object.

## Parameters

| Name | Description |
|---|---|
| handle | Reference of the map object. |
| visible | 0 = not visible 1 = visible |
