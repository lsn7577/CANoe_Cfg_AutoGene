# CarMaker_StopTestSeries

> Category: `CarMaker` | Type: `function`

## Syntax

```c
long CarMaker_StopTestSeries( );
```

## Description

Stops test series immediately in CarMaker.

## Return Values

APO return code

## Example

```c
// stop test series
gErrorState = CarMaker_StopTestSeries();
```
