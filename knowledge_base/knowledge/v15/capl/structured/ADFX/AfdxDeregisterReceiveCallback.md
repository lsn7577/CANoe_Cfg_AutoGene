# AfdxDeregisterReceiveCallback

> Category: `ADFX` | Type: `function`

## Syntax

```c
long AfdxDeregisterReceiveCallback( char callback[] );
```

## Description

Use this function to deregister a CAPL callback that has been registered with AfdxRegisterReceiveCallback.

## Parameters

| Name | Description |
|---|---|
| callback | Name of CAPL callback to be deregistered. |
