# FRSOnStopped

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSOnStopped(char CallbackName[]);
```

## Description

Registers a callback. The callback will be executed when the session ends. The callback function must be added in the CAPL section callback.

## Parameters

| Name | Description |
|---|---|
| CallbackName | Any desired name can be used as name of the callback function. |
