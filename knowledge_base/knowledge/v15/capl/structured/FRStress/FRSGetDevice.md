# FRSGetDevice

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSGetDevice();
```

## Description

Indicates the device number of the FRstress hardware, set with FRSSetDevice.

## Return Values

Returns the device number of the currently active hardware unit.
