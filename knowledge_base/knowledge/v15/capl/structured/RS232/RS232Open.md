# RS232Open

> Category: `RS232` | Type: `function`

## Syntax

```c
dword RS232Open( dword port );
```

## Description

Opens a serial port.

## Parameters

| Name | Description |
|---|---|
| port | A number between 1 and 255 identifying a serial port. |

## Example

```c
if ( 1==RS232Open(1) ) write(“It works with port 1.”);
```
