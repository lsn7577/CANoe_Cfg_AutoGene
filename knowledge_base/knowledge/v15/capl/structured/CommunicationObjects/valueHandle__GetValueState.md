# valueHandle::GetValueState

> Category: `CommunicationObjects` | Type: `method`

## Syntax

```c
dword valueHandle::GetValueState();
```

## Description

Returns the state of the value, which tells in particular whether it has already been updated once since the last call to valueHandle::ResetValueState.

## Example

```c
long ret;
consumedEventRef * anEvent;
anEvent = lookupConsumedEvent(path);
if (anEvent.GetValueState() != 3) // not yet received
{
  anEvent.ClearUpdateFlag();
  ret = testWaitForUpdateFlag(anEvent, 200);
  // ...
}
```
