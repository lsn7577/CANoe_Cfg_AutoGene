# xcpIsConnected

> Category: `XCP` | Type: `function`

## Syntax

```c
long xcpIsConnected (char ecuQualifier[]);
```

## Description

Returns the current connection status of a XCP/CCP device.

## Parameters

| Name | Description |
|---|---|
| ecuQualifier | Name of the device – configured within the CANoeXCP/CCP Window. |
