# CANstressSetBTR

> Category: `CANstress` | Type: `function`

## Syntax

```c
void CANstressSetBTR (dword btr1, dword btr2);
```

## Description

Sets the value of the Bus Timing Register. This setting affects the baud rate of the CAN bus set in CANstress. The CAN controller data sheet will list valid pairs of values.

## Parameters

| Name | Description |
|---|---|
| btr0 | New value for BTR0. |
| btr1 | New value for BTR1. |

## Return Values

—
