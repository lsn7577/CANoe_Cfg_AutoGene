# CANstressOpen

> Category: `CANstress` | Type: `function`

## Syntax

```c
void CANstressOpen( char fileName[] );
```

## Description

Opens a CANstress configuration file.

## Parameters

| Name | Description |
|---|---|
| fileName | If the passed configuration file is configured as a User File, only the file name is passed here. Otherwise either an absolute path or a relative path relating to the folder of the CANoe configuration must be passed. |

## Return Values

—
