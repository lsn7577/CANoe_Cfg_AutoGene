# a429CheckParity

> Category: `A429` | Type: `function`

## Syntax

```c
dword a429CheckParity( a429word myA429word, int parity_value );
```

## Description

Checks the parity over 32 bits of the specified ARINC word.

## Parameters

| Name | Description |
|---|---|
| myA429word | ARINC word |
| parity_value | specify whether check is odd or even 2: odd parity 3: even parity |
