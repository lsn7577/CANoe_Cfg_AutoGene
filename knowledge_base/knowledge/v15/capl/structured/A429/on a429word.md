# on a429word

> Category: `A429` | Type: `event`

## Description

The event procedure on a429word is called on every event, related to an error free ARINC word. Additional information related to this event is available via the selectors. The key word this is available within an on a429word procedure, to access the data of the ARINC word event that has just been received.
