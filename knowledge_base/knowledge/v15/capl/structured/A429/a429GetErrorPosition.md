# a429GetErrorPosition

> Category: `A429` | Type: `function`

## Syntax

```c
long a429GetErrorPosition( a429worderror myA429word );
```

## Description

For every Rx error an error position is provided. You may access this information with the operator this in an on a429Worderror handler.

## Parameters

| Name | Description |
|---|---|
| myA429word | ARINC word in on a429Worderror handler |
