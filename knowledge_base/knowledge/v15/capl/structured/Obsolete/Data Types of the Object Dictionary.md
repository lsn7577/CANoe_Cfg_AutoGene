# Data Types of the Object Dictionary

> Category: `Obsolete` | Type: `notes`

## Description

The following table shows the non-structured data types supported by the CANopen node layer.Manufacturer-specific non-structured data types are also supported but not listed in the table.

See Also

| Index | Name | Length |
|---|---|---|
| 0x01 | Boolean | 1 Bit |
| 0x02 | Integer8 | 8 Bit |
| 0x03 | Integer16 | 16 Bit |
| 0x04 | Integer32 | 32 Bit |
| 0x05 | Unsigned8 | 8 Bit |
| 0x06 | Unsigned16 | 16 Bit |
| 0x07 | Unsigned32 | 32 Bit |
| 0x08 | Real32 | 32 Bit |
| 0x09 | Visible String | variable (0..536,870,911 byte) |
| 0x0A | Octet String | variable (0..536,870,911 byte) |
| 0x0C | Time of Day | 48 Bit |
| 0x0D | Time Difference | 48 Bit |
| 0x0F | Domain | variable (0..536,870,911 byte) |
| 0x10 | Integer24 | 24 Bit |
| 0x11 | Real64 | 64 Bit |
| 0x12 | Integer40 | 40 Bit |
| 0x13 | Integer48 | 48 Bit |
| 0x14 | Integer56 | 56 Bit |
| 0x15 | Integer64 | 64 Bit |
| 0x16 | Unsigned24 | 24 Bit |
| 0x18 | Unsigned40 | 40 Bit |

| 0x1A | Unsigned56 | 56 Bit |
