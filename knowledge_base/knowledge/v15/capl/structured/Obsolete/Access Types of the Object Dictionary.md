# Access Types of the Object Dictionary

> Category: `Obsolete` | Type: `notes`

## Description

The permissible access types are coded as follows:

See Also

| Code | Description |
|---|---|
| 0 | Only read |
| 1 | Read and write |
| 2 | Only write |
| 3 | Read and write - can only be mapped in a RPDO |
| 4 | Read and write - can only be mapped in a TPDO |
| 5 | Only read - can only be mapped in a TPDO |
| 6 | Only write - can only be mapped in a RPDO |
| 7 | Is handled in the application |
