# traceWindowClear

> Category: `Other` | Type: `function`

## Syntax

```c
void traceWindowClear(char[] windowName);
```

## Description

Clears the contents of the Trace Window.

## Parameters

| Name | Description |
|---|---|
| windowName | The name of the Trace Window. |
