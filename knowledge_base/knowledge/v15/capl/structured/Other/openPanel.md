# openPanel

> Category: `Other` | Type: `function`

## Syntax

```c
void openPanel(char panelName[]);
```

## Description

Opens a panel.

The panel is accessed by its individual panel name that is entered in the Panel Designer.

## Parameters

| Name | Description |
|---|---|
| panelName | Panel name, restricted to 128 characters. |

## Return Values

—
