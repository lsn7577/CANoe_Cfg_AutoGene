# SetVideoOfflineLoggerCam

> Category: `Other` | Type: `function`

## Syntax

```c
SetVideoOfflineLoggerCam(char windowName[], long camType, [char filePrefix[]]);
```

## Description

Sets the offline mode logger cam (type and prefix) for a CANoeVideo Window.

## Parameters

| Name | Description |
|---|---|
| windowName | The name of the Video Window. |
| camType | The type of logger cam: 0\|cCamlog2 1\|cHostCam For details see the Image Files Properties in the CANoeVideo Configuration dialog. |
| filePrefix | The corresponding prefix of the image files generated by the cam. For cCamlog2 this can be omitted. For details see the Image Files Properties in the CANoeVideo Configuration dialog. |

## Return Values

—

## Example

```c
on preStart
{
  //Configure offline source (image files)
  SetVideoOfflineSource("Video Window", 2, "..\\Measurement\\VideoFrames");

  //Configure pattern to parse the time stamps of the images
  SetVideoOfflineLoggerCam("Video Window", 1, "CAM1_");
}
```
