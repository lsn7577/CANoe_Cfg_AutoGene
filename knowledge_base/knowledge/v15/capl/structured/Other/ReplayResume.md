# ReplayResume

> Category: `Other` | Type: `function`

## Syntax

```c
dword ReplayResume (char pName[]);
```

## Description

Starts the Replay Block with the name pName after it is suspended by ReplaySuspend.

## Parameters

| Name | Description |
|---|---|
| pName | Name of the Replay Block. |
