# random

> Category: `Other` | Type: `function`

## Syntax

```c
dword random(dword x);
```

## Description

Calculates a random number between 0 and x-1.

## Return Values

Random number between 0 and x-1.

## Example

```c
dword x;
// generate random number in the interval [0;100]
x = random(100);
```
