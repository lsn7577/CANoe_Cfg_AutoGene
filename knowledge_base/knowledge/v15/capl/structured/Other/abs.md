# abs

> Category: `Other` | Type: `function`

## Syntax

```c
long abs(long x);
double abs(double x);
```

## Description

Returns the absolute value.

## Return Values

Absolute value of x.

## Example

```c
long x;
x = abs(15); // Result: 15
```
