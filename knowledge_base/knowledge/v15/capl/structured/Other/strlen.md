# strlen

> Category: `Other` | Type: `function`

## Syntax

```c
long strlen(char s[]);
```

## Description

The functional result is the length of the string s in bytes.

## Return Values

Length of the string in bytes.

## Example

Result:

length = 9

```c
long length;
char buffer[100] = "CANalyzer";
length = strlen(buffer);
```
