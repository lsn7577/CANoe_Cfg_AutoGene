# _pow

> Category: `Other` | Type: `function`

## Syntax

```c
double _pow(double x, double y);
```

## Description

Returns x to the power of y.

## Parameters

| Name | Description |
|---|---|
| x | base |
| y | exponent |

## Return Values

x to the power of y.

## Example

```c
double result;
result = _pow(2.0, 3.0); // result == 8.0
result = _pow(4.0, 0.5); // result == 2.0
```
