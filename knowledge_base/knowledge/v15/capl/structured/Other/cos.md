# cos

> Category: `Other` | Type: `function`

## Syntax

```c
double cos(double x);
```

## Description

Calculates cosine of x.

## Return Values

Cosine of x.

## Example

or

```c
double x;
x = cos(PI); // result -1
double tangens(double x) {
return sin(x) / cos(x);
}
```
