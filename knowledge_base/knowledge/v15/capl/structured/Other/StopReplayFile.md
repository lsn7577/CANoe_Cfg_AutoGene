# StopReplayFile

> Category: `Other` | Type: `function`

## Syntax

```c
void StopReplayFile(dword handle);
```

## Description

Stops the replay file from playing with the handle handle.

## Parameters

| Name | Description |
|---|---|
| handle | Handle of the started replay file. The handle is returned from the StartReplayFile function when starting the reply file. |

## Return Values

—
