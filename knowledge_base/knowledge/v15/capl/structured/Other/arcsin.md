# arcsin

> Category: `Other` | Type: `function`

## Syntax

```c
double arcsin(double x);
```

## Description

Calculates arcsine of x.

## Parameters

| Name | Description |
|---|---|
| x | Value between –1 and 1 whose arcsine is to be calculated. Values outside this range cause a CAPL runtime error. |

## Return Values

Arcus Sine of x.

## Example

```c
double x;
x = arcsin(1); // result PI/2
x = arcsin(0); // result 0
```
