# ILFaultInjectionEnableMsg

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILFaultInjectionEnableMsg (dbMsg msg)
```

## Description

Enables the sending of the message.

## Parameters

| Name | Description |
|---|---|
| msg | Message that should be re-enabled after having disabled it with ILFaultInjectionDisableMsg. |
