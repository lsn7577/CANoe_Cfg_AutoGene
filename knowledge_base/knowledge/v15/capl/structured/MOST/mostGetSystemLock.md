# mostGetSystemLock

> Category: `MOST` | Type: `function`

## Syntax

```c
long mostGetSystemLock(long channel);
```

## Description

Returns the current state of the System Lock Flag.

## Parameters

| Name | Description |
|---|---|
| channel | Application channel number. |
