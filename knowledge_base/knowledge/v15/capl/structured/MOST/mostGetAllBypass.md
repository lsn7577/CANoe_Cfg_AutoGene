# mostGetAllBypass

> Category: `MOST` | Type: `function`

## Syntax

```c
long mostGetAllBypass(long channel);
```

## Description

Return indicates whether the MOST hardware bypass is closed (Return value = 1) or open (Return value = 0).

## Parameters

| Name | Description |
|---|---|
| channel | Channel of the connected hardware. |
