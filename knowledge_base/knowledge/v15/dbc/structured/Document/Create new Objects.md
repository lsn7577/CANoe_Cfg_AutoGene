# Create new Objects

> Category: `DBC` | Subcategory: `Document` | Type: `concept`

New objects can be created by several commands of menu Edit, by context menus or by the toolbar.

Possible tree objects are:

Object type

Description

Objects

CANopen objects

Sub Objects

CANopen sub objects

Modules

Allow building of flexible devices which can be extended by modules

Name Sections

For compact sub objects it is possible to assign explicit names, if the default names are not useful enough. For this a list of according names can be created.

Object Links

In order to ease the implementation of a configuration tool it is possible to group related objects together via an object link.

Dynamic Channels

Support of intelligent or programmable devices

Tools

Describes some aspects of the usage of the EDS by software packages (specified in CiA405)

Sections

Allows additional sections with user-defined entries

Copy and Paste | Multiple Selection | Copy with offset
