# Command Edit | New Name Section

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

For compact sub objects it possible to assign explicit names, if the default names are not useful enough.

If a main object with the compact sub objects is selected in the EDS Window a new name section for this object is created. Otherwise a dialog pops up and the index of the target object has to be chosen from a list of all existing main objects with compact sub objects.

After creation the attribute window displays a list of all compact sub objects and allows to set names.
