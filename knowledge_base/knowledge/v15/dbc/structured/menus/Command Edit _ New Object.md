# Command Edit | New Object

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Open a dialog to create a new main object and possibly sub objects. The object is inserted into the EDS tree.

Symbol:

Step by Step Procedure for creating a new object

| Step by Step Procedure for creating a new object Set index of the new main object. If an object with the same index is contained in the database the values of the database object are adopted Decide if new object has to be an extended or fixed object. For such objects it is necessary to press button [Next] of the dialog and to select the number of the module or to create a new module. If there are manufacturer or device profile specific data types with object type DEFSTRUCT one of these structures can be selected in the section Creation of objects. On creation of the object the sub objects, which are described in the structure, are created too. Finish creation. |
