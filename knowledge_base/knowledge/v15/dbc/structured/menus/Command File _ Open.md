# Command File | Open

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Opens an existing file (EDS, DCF, XDD and XDC) in a new window.

You can open multiple files at once, for switching among the files us the command Window. It is also possible to open a file via drag & drop from the Windows Explorer to the CANeds Window. See command Recent Files for opening recently opened files.

Symbol:

Shortcut: <Ctrl>+<O>
