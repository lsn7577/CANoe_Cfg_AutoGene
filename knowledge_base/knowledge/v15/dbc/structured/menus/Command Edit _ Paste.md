# Command Edit | Paste

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Create one or more new tree objects with data of the Windows clipboard. The object is added to the group directly after the selected tree object in the EDS Window.

If the object already exists the dialog Duplicate Object appears. The preferred action (overwrite or leave existing object) has to be selected.

Symbol:

Shortcut: <Ctrl>+<V> or <Shift>+<INS>
