# Command File | Save As

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Use this command to save and name the active file. CANeds displays the Save As dialog so you can name your file and select the format of it. If a EDS/DCF is saved as XDD/XDC, information about modules are lost since it cannot described in XDDs/XDCs.

If the file is saved as a DCF or XDC, in the Device Commissioning dialog device specific information can be set such as node ID, node name, baud rate, the network number and the LSS serial number. Additionally it can be defined if the device can be used as CANopen Manager.

To save a file with its existing name and directory and in the same format, use the command File | Save.
