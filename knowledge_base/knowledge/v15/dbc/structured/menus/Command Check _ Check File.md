# Command Check | Check File

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Check the active file with CANchkEDS. The results of the check are shown in the output window.

If the file is a XDD or XDC, a check to the CANopen schema is done first (the path of the schema can be specified in the settings). After that the file is exported temporarily in an EDS or DCF, which is checked with CANchkEDS afterwards.

Symbol:

Shortcut: <F5>
