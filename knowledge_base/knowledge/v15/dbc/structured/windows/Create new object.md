# Create new object

> Category: `DBC` | Subcategory: `windows` | Type: `concept`

In this dialog first the index of the new object has to be edited. Pressing [Finish] creates the object.

If there are complex manufacturer or device profile specific data types in the EDS, one of these data types can be selected in the section Creation of sub objects. On creation the number of sub objects and the data types of the sub objects are taken over by the selected data type.

If the object shall be added to a module the appropriate type (fixed or sub extension) has to be selected. Then button [Next], which is now enabled, is clicked. Before the object is created by clicking [Finish] the number of the module has to be selected.
