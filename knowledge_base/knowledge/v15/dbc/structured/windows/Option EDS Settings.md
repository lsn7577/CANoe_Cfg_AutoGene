# Option EDS Settings

> Category: `DBC` | Subcategory: `windows` | Type: `concept`

This page contains the settings regarding to the EDS properties.

## Type Definitions

In CiA301 it is not specified in which section manufacturer and device profile specific data types are stored. Therefore a section has to be specified, which is either one of the existing ones (OptionalObjects or ManufacturerObjects) or an arbitrary name.

Specific data types are named in the page Type Definitions in the main window. To show all data types in the object dictionary window, the corresponding check mark has to be set.
