# Sub extension object attributes

> Category: `DBC` | Subcategory: `object_attributes` | Type: `concept`

On this page the attributes of a sub extension object are shown. Sub extension objects that instantiate new sub objects.

There are two additional fields according to the sub object attributes:

Created per Module

Number of sub objects with this attributes that are created per module or number of bits that are created per module to build a new sub object..

Maximum Sub Index

If an array is filled up to the end, the next object may be used to create the remaining sub objects. This value defines the maximum sub index after which the next object is used. If the value is 0 or the entry is missing, the next object can not be used.
