# File

> Category: `Panel` | Subcategory: `Ribbon` | Type: `concept`

File

Home

Panel

Properties

New

With a click on the symbol you open a new panel.

Open

With a click on the symbol you open the Windows Explorer to select a previously saved panel file.

With a click on you open the recent panel file list.

Import

With a click on the symbol you open the Open dialog. In this dialog, you can select and import an existing panel file.

Save

With a click on the symbol you save the panel.The Save As dialog automatically opens when the panel is saved for the first time.

Save As...

With a click on the symbol you open the Save As dialog.

Save All

With a click on the symbol you save all opened and changed panels.

Add to Configuration

With a click on the symbol you open the Save As dialog. In this dialog you can add the panel to the configuration.

Options

In the Options dialog you can configure global settings for controls, panels and the grid.

With a click on the symbol you open the Options dialog.

Close

With a click on the symbol you close the active panel.

If you have not yet saved all changes to the open panel, you will be prompted to do so automatically. Select the appropriate option in the window which appears.

Close All

With a click on the symbol you close all open panels.

If you have not yet saved all changes to the open panels, you will be prompted to do so automatically. Select the appropriate option in the window which appears.

Help

With a click on the symbol you open the help.

About

With a click on the symbol you open a dialog that displays program version number, individual serial number and name of the licensee.

Exit

Exits the Panel Designer.
