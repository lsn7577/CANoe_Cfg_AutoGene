Note

A change in the alignment affects all opened panels.

The selected alignment is saved and reused when the Panel Designer is reopened.

## Control Alignment

## Controls snap to snaplines

With this option, snaplines are displayed as soon as a control is moved close to the panel border or close to another control. Thus, for example, you can configure all elements to be spaced equally from the panel border or all elements to be aligned with one another.

The snaplines also allow you to easily adapt the sizes of the various controls to one another.

Note

You can temporarily skip the automatic alignment with the shown snaplines by pressing <Alt> when moving the element.

A grid is not displayed in this mode.

## Controls snap to grid

With this option, you can align the controls to a grid. The upper left corner of the control is the "handle" for this. You can define the spacing of the grid points in the X- and Y-directions in the Grid size. The smallest possible value is 2 point, and the largest possible value is 100 point.

When the size of a control is defined, the closest grid line (horizontal and vertical) is always used as the "handle".

## Controls free positionable

With this option, you can freely define the position and size of a control; an alignment to snaplines or grid points does not occur.

The grid points can be displayed and hidden same as with the Controls snap to grid option.

## Grid Settings

Here you enter the distance of the dots (width and height).

With Show Grid you switch the display of the dots on and off.

These settings are not available with the Controls snap to snaplines alignment.
