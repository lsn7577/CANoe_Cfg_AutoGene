# Save Panel

> Category: `Panel` | Subcategory: `Ribbon` | Type: `concept`

In the Save As dialog you can save the panel with a new name or in a new directory.

This dialog automatically opens when the panel is saved for the first time, where the name and storage location of the panel can be entered.

## Add Panel to Configuration

With this activated option you can integrate your created panels into the currently loaded CANoe/CANalyzer configuration.

## Use File Name as Panel Name

With this activated option the panel file automatically get the name of the panel that should be saved.
