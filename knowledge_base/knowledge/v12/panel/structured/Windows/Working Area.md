# Working Area

> Category: `Panel` | Subcategory: `Windows` | Type: `concept`

You can create or modify panels in the working area.

To create a new panel, open a new panel by selecting File ribbon tab|New. Only once you have done this will you be able to assign controls and symbols to the panel.

Every element (panel and controls) selected in the working area can be configured in the Properties Window.

To facilitate alignment of the controls, snaplines on which controls can be aligned appear when you start the assignment process.

Note

You can temporarily skip the automatic alignment with the shown snaplines by pressing <Alt> when moving the element.

In addition to the guide lines, the Home ribbon tab features additional edit options.

You can edit a number of panels in parallel in the Panel Designer. In the working area, each panel is opened in a separate tab.

Once a control has been selected, marker points will appear which you can use to adjust the size in accordance with your requirements.

The panel's context menu features options for opening new or existing panels and saving the open panel.
