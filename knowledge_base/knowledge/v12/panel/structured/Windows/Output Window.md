# Output Window

> Category: `Panel` | Subcategory: `Windows` | Type: `concept`

You can see information and error messages relating to the currently active panel in the Output Window.

A check is carried out to determine if all controls are linked with symbols (if possible) and if all these symbols are present in the current database when loading a panel. If a control has no symbol link or a symbol is not found in the database, the Output Window is opened automatically with a corresponding message. Additional missing symbol links are added line-by-line.

## Selecting a message

If you select a message in the Output Window, the corresponding control is also selected in the panel.

## Deleting the messages in the Output Window

Select the Delete All menu command in the context menu of the Output Window to delete all messages in the Output Window

## Updating the Output Window

The Output Window is updated when a panel is loaded and when a symbol check is selected.

In the case of a database or panel change, the Output Window is not updated automatically.
