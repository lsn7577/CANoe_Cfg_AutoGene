# Menu Check

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

This menu allows you to check parts or the whole EDS with CANchkEDS. Check reports are written to the output window. Via context menu of the output window the help topic to the selected error or warning notification can be called. It is also possible to jump to the tree object which is responsible for the notification.
