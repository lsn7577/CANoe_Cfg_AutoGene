# Command Context Help (Toolbar)

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Use the Context Help command to obtain help on some portion of the application. When you choose the context help button of the toolbar, the mouse pointer will change to an arrow and question mark. Then click somewhere in the application window, such as another toolbar button. The help topic will be shown for the item you clicked.

Symbol:

Shortcut: <Shift>+<F1>
