# Command Database | Edit List

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Opens the database dialog to modify the list of database files. Files can be added, removed or moved.

The order of the files in the dialog is important because identical objects existing in more than one files are overwritten. Only the multiple object in the last file appears in the database window.

Symbol:

Shortcut: <Ctrl>+<F2>
