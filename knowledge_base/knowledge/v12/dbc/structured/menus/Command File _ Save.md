# Command File | Save

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Use this command to save the active file to its current name and directory.

When you save a file for the first time, CANeds displays the Save As dialog so you can name your file. If you want to change the name and directory of an existing file before you save it, choose the command File | Save As.

Symbol:

Shortcut: <Ctrl>+<S>
