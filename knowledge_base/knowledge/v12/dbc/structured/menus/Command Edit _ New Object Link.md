# Command Edit | New Object Link

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

In order to ease the implementation of a configuration tool it is possible to group related objects together.

If a main object is objects is selected in the EDS Window the index of this object is the index of the new object link. Otherwise the index has to be chosen out of a list of all existing main objects. The following dialog presents a list of all existing main objects.

After creation the attribute window displays a list of all existing main objects. All objects marked by a check mark are linked.
