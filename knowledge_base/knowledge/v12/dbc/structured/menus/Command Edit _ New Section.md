# Command Edit | New Section

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Create a new section.

Symbol:

The application explicitly allows the usage of additional sections. It does so for future compatibility and for the purpose of manufacturer specific extensions. Since the basic format of an EDS is the Windows INI file format, additional sections normally will not disturb the behavior of any software, that reads the EDS.

Entries in the new section have to be entered in the Windows INI file format.

Caution!

The naming of those additional sections must not interfere with standardized section names. It has to be considered, that naming may interfere with future extensions of CiA301.
