# Command Edit | New Sub Object

> Category: `DBC` | Subcategory: `menus` | Type: `concept`

Open a dialog to create a new sub object. The sub object is inserted into the EDS tree.

Symbol:

Step by Step Procedure for creating a new sub object

| Step by Step Procedure for creating a new sub object Set the index of the main object. If a main object is selected in the EDS Window the index of this object is the default value. Set the number of the new sub object. If an object with the same index and sub index is contained in the database the values of the database object are adopted. Otherwise some default values are set. Finish creation. |
|---|
