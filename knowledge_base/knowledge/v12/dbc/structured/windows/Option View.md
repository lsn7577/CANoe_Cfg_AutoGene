# Option View

> Category: `DBC` | Subcategory: `windows` | Type: `concept`

This page contains the settings regarding to the view of the tool.

## Output Window

If a file is opened or checked the output window scrolls to show the latest notification. This is also done if a selected object loses its focus and an error is discovered. To avoid the scrolling in the second case it may be switched off.

For object check with command Check EDSthe automatic scroll can not be switched off.

## Window Layout

To restore the layout of the different windows when the next EDS is opened, this check box can be activated.

It is also possible to save the layout by the command Save Window Layout of the View menu.

## Database Window

If a database with PDOs and SDOs is loaded it may be useful to reduce the number of visible PDOs and SDOs in the database window. There are two fields to set the maximum number.

Although some objects are not visible they are still used in check procedures.

If the check box Show complete database file path is selected then the file path is shown in the file view (see command File View of menu Database). Otherwise only the file name is shown.
