# containsKey

> Category: `AssociativeFields` | Type: `function`

## Syntax

```c
m.containsKey(3));
```

## Description

Returns the information if a key is available in the field.

## Parameters

| Name | Description |
|---|---|
| key | Key that should be checked. |

## Return Values

1 if the key was available - otherwise 0

## Complexity

O(log(N))

## Example

```c
if (m.containsKey(3)) { // …
```
