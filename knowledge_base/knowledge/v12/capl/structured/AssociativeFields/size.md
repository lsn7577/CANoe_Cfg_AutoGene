# size

> Category: `AssociativeFields` | Type: `function`

## Syntax

```c
m.size() == 0) write("Map is empty!");
```

## Description

Returns the number of elements in the field.

## Return Values

Number of elements in the field.

## Example

```c
if (m.size() == 0) write("Map is empty!");
```
