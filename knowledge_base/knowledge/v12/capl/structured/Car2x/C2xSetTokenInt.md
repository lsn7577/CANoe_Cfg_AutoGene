# C2xSetTokenInt

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetTokenInt( long packet, char protocolDesignatorl[], char tokenDesignator[], long value ); // form 1
```

## Description

This function sets the integer value of a token.

Form 2 with byte offset sets a part of the token data as integer.

## Return Values

0 or error code
