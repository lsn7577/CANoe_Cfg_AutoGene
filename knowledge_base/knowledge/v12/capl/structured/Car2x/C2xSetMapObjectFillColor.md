# C2xSetMapObjectFillColor

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetMapObjectFillColor( long handle, float fillColor );
```

## Description

Sets the fill color of a map object or the text color of a text object.

## Return Values

0 if success, else the set went wrong.
