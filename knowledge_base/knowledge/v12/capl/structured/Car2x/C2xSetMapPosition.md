# C2xSetMapPosition

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetMapPosition( float latitude, float longitude );
```

## Description

Sets the map to a specific position.

If the Auto-Follow option is enabled for a vehicle, the option will be disabled.

## Return Values

0 if success, else the set went wrong
