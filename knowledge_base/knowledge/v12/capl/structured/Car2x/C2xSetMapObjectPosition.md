# C2xSetMapObjectPosition

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetMapObjectPosition( long handle, float latitude1, float longitude1 ); // form 1
```

## Description

Sets the GPS position of a map object either for one (form 1) or two (form 2) points.

## Return Values

0 if success, else the set went wrong
