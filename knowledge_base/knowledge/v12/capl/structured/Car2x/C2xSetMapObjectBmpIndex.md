# C2xSetMapObjectBmpIndex

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetMapObjectBmpIndex( long handle, long bitmapIndex );
```

## Description

Sets the index of a bitmap in a multi bitmap file for a map object of the type bitmap which must be shown.

## Return Values

0 if success, else the set went wrong
