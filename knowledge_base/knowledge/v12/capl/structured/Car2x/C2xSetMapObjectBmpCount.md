# C2xSetMapObjectBmpCount

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetMapObjectBmpCount( long handle, long bitmapCount );
```

## Description

Sets the number of bitmaps in a multi bitmap file of a map object.

## Return Values

0 if success, else the set went wrong.
