# C2xSetMapObjectSize

> Category: `Car2x` | Type: `function`

## Syntax

```c
long C2xSetMapObjectSize( long handle, float size ); // form 1
```

## Description

Sets either the size (form 1) or the height and width (form 2) of a map object.

## Return Values

0 if success, else the set went wrong
