# AvbSetMediaType

> Category: `IP` | Type: `function`

## Syntax

```c
long AvbSetMediaType(dword talkerHandle, dword mediaTypeHandle);
```

## Description

Sets the media type for a Talker. Setting the media type means the Talker can expects data that conforms to that media type passed to its AvbSend function.

## Return Values

0: The function completed successfully.
