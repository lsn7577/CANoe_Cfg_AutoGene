# AvbOpenListener

> Category: `IP` | Type: `function`

## Syntax

```c
dword AvbOpenListener( );
```

## Description

The function creates a Listener for use in connection-based, message-oriented communications.

## Return Values

0: The function failed. Call AvbGetLastError to get a more specific error code.
