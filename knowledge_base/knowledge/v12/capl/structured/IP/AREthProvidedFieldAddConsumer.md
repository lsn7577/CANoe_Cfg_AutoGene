# AREthProvidedFieldAddConsumer

> Category: `IP` | Type: `function`

## Syntax

```c
long AREthProvidedFieldAddConsumer(dword pfHandle, dword remoteIPv4Address, dword remoteUDPPort, dword remoteTCPPort);
```

## Description

Adds a consumer to a provided field. Afterwards the field notifications will be sent to this consumer.

## Return Values

0: The function was successfully executed
