# AREthProvidedFieldRemoveConsumer

> Category: `IP` | Type: `function`

## Syntax

```c
long AREthProvidedFieldRemoveConsumer(dword pfHandle, dword remoteIPv4Address, dword remoteUDPPort, dword remoteTCPPort);
```

## Description

Removes a consumer from a provided field that has been added by AREthProvidedFieldAddConsumer.

## Return Values

0: The function was successfully executed
