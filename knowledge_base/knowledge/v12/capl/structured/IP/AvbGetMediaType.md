# AvbGetMediaType

> Category: `IP` | Type: `function`

## Syntax

```c
dword AvbGetMediaType(dword listenerOrTalkerHandle);
```

## Description

Gets the media type for a Listener or a Talker.

## Return Values

0: The function failed. Call AvbGetLastError to get a more specific error code.
