# AvbCloseTalker

> Category: `IP` | Type: `function`

## Syntax

```c
dword AvbCloseTalker(dword talkerHandle);
```

## Description

The function closes the Talker. Upon successful completion the Talker refer-enced by the passed handle is no longer valid.

## Return Values

0: The function completed successfully.
