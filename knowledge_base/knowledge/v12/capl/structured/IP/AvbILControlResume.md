# AvbILControlResume

> Category: `IP` | Type: `function`

## Syntax

```c
dword AvbILControlResume();
```

## Description

Resumes AVB/TSN automatic message sending behavior after a previous suspension by AvbILControlWait.

Previously opened Talkers and Listeners do not have to be opened again.

## Return Values

0: The function was successfully executed.
