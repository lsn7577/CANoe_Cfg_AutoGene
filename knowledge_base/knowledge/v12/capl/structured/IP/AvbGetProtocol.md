# AvbGetProtocol

> Category: `IP` | Type: `function`

## Syntax

```c
dword AvbGetProtocol(dword listenerOrTalkerHandle, dword& retProtocol);
```

## Description

The function retrieves the AVTP protocol of the Listener or Talker.

## Return Values

0: The function completed successfully.
