# GNSSStopSimulation

> Category: `J1939` | Type: `function`

## Syntax

```c
dword GNSSStopSimulation();
```

## Description

This function stops the simulation. This does not result in waypoints that are set being deleted.

## Return Values

error code

## Example

```c
GNSSStopSimulation();
```
