# J1939TestChkQuery_NumEvents

> Category: `J1939` | Type: `function`

## Syntax

```c
long J1939TestChkQuery_NumEvents( long checkId );
```

## Description

Returns the number of events generated by this check since its initialization (reset).

## Example

```c
long cycleCheck;
cycleCheck = J1939TestChkCreate_AbsCycleTimeViolation( 1, 2, 0xf123, 0.8, 1.2, "CycleCallback" );

J1939TestChkControl_Start( cycleCheck );
//...
write( "Check NumEvents=%d", J1939TestChkQuery_NumEvents(cycleCheck));
```
