# J1939GetLastError

> Category: `J1939` | Type: `function`

## Syntax

```c
dword J1939GetLastError( void );
```

## Description

This function returns the last error code.

## Return Values

error code

## Example

```c
dword errCode;
errCode = J1939GetLastError();
```
