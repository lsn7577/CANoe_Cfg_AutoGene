# J1939ILEnableMsg

> Category: `J1939` | Type: `function`

## Syntax

```c
long J1939ILEnableMsg(dbMessage dbMessage); // form 1
```

## Description

Re-enables the sending of the message after having disabled it with ILFaultInjectionDisableMsg/J1939ILDisableMsg.
