# KLine_SetW5Tester

> Category: `KLine` | Type: `function`

## Syntax

```c
long KLine_SetW5Tester( dword W5_us)
```

## Description

Sets the W5 timing befor the tester starts to transmit the address byte.

## Return Values

On success 0, otherwise a value less than 0.
