# KLine_SetP3Tester

> Category: `KLine` | Type: `function`

## Syntax

```c
long KLine_SetP3Tester(dword P3_us)
```

## Description

Sets the P3 time which the tester waits until transmitting a further request.

## Return Values

On success 0, otherwise a value less than 0.
