# KLine_SetW4Tester

> Category: `KLine` | Type: `function`

## Syntax

```c
long KLine_SetW4Tester( dword W4_us)
```

## Description

Sets the W4 timing of the tester between keybyte2 and keybyte2Not.

## Return Values

On success 0, otherwise a value less than 0.
