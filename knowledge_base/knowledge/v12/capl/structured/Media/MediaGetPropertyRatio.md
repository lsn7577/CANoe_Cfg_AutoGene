# MediaGetPropertyRatio

> Category: `Media` | Type: `function`

## Syntax

```c
long MediaGetPropertyRatio(dword objHandle, char propertyName[], dword& numerator, dword& denominator);
```

## Description

Retrieves a property whose value is a ratio. Properties can be retrieved on an object specific basis for a media type object (e.g. returned by MediaGetMediaType).

## Return Values

0: The function succeeded.
