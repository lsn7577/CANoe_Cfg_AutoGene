# MediaCreateSourceReaderFromURL

> Category: `Media` | Type: `function`

## Syntax

```c
dword MediaCreateSourceReaderFromURL(char mediaFileUrl[]);
```

## Description

Creates the source reader from a URL.

For more information regarding the source reader see source reader.

## Return Values

0: The function failed. Call MediaGetLastError to get a more specific error code.
