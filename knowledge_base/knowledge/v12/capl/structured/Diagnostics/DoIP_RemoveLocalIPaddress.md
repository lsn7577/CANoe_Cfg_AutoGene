# DoIP_RemoveLocalIPaddress

> Category: `Diagnostics` | Type: `function`

## Syntax

```c
long DoIP_RemoveLocalIPaddress(char ipAddress[]);
```

## Description

Removes an IP address the vehicle will no longer listen at.
