# CANstressIsPending

> Category: `CANstress` | Type: `function`

## Syntax

```c
long CANstressIsPending();
```

## Description

Serves to query whether the CANstress hardware is in the state Pending.

## Return Values

1: If the CANstress hardware is in the state Pending.
