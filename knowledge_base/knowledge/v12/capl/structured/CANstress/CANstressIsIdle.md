# CANstressIsIdle

> Category: `CANstress` | Type: `function`

## Syntax

```c
long CANstressIsIdle();
```

## Description

Serves to query whether the CANstress hardware is in the state Idle.

## Return Values

1: If the CANstress hardware is in the state Idle.
