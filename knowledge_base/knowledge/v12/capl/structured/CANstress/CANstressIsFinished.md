# CANstressIsFinished

> Category: `CANstress` | Type: `function`

## Syntax

```c
long CANstressIsFinished();
```

## Description

Serves to query whether the CANstress hardware is in the state Finished.

## Return Values

1: If the CANstress hardware is in the state Finished.
