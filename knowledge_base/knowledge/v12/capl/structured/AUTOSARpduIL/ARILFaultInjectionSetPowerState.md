# ARILFaultInjectionSetPowerState

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILFaultInjectionSetPowerState (long value);
```

## Description

Activates or deactivates the power status locally. Modifying the power state of the local IL instance is similar to calling ARILControlSimulationOn or ARILControlSimulationOff.

## Return Values

0: No error.
