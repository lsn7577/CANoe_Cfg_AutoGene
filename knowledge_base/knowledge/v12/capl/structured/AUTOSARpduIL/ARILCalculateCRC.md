# ARILCalculateCRC

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILCalculateCRC (char aMsgName[], cahr sigGroupName[], BYTE payload[], dword aPayloadLength, dword crc[]);
```

## Description

Calculates the CRC of the PDU according to the given payload.

## Return Values

0: No error.
