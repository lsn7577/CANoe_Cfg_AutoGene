# ARILFaultInjectionDisablePDU

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILFaultInjectionDisablePDU (dbMessage);
```

## Description

Disables the sending of the PDU except by calling the function ARILSetPDUEvent.

## Return Values

0: No error.
