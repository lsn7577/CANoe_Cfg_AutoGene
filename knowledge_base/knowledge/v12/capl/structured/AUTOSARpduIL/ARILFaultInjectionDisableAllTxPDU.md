# ARILFaultInjectionDisableAllTxPDU

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILFaultInjectionDisableAllTxPDU ( );
```

## Description

Disables the sending of all PDUs of the current node instance.

## Return Values

0: No error.
