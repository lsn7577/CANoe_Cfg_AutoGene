# ARILGetSigGroupCount

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILGetSigGroupCount (char aMsgName[]);
```

## Description

Evaluates how many signal groups are defined inside the PDU.

## Return Values

≥ 0: The number of signal groups.
