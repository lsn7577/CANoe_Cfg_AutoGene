# ARILGetSigGroupName

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILGetSigGroupName (char aMsgName[], dword index, char sigGroupName[], long maxSigGroupNameLen);
```

## Description

Retrieves the name of the nth signal group of the PDU.

## Return Values

0: No error
