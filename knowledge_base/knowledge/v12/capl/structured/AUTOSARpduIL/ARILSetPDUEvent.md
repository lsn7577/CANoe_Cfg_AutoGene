# ARILSetPDUEvent

> Category: `AUTOSARpduIL` | Type: `function`

## Syntax

```c
long ARILSetPDUEvent (dbMessage);
```

## Description

A call of this function will lead to a transmission of the associated PDU. This function considers GenMsgDelayTime.

## Return Values

0: No error.
