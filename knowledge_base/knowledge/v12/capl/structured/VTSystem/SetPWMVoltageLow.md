# SetPWMVoltageLow

> Category: `VTSystem` | Type: `function`

## Syntax

```c
long SysVarNamespace.SetPWMVoltageLow (double Voltage);
```

## Description

Specifies the low level on output of a digital output signal, especially a PWM signal.

## Return Values

0: Successful call
