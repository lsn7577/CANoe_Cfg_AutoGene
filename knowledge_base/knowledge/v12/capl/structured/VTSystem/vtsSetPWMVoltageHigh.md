# vtsSetPWMVoltageHigh

> Category: `VTSystem` | Type: `function`

## Syntax

```c
long vtsSetPWMVoltageHigh (char Target[], double Voltage);
```

## Description

Specifies the high level on output of a digital output signal, especially a PWM signal.

## Return Values

0: Successful call
