# _ceil

> Category: `Other` | Type: `function`

## Syntax

```c
float _ceil(float x);
```

## Description

Calculates the ceiling of a value, i.e. the smallest integer larger or equal to the value.

## Return Values

Ceiling of x.

## Example

```c
float x;
x = _ceil(3.6); // x == 4.0
```
