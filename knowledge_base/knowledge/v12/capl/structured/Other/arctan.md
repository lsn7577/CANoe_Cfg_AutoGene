# arctan

> Category: `Other` | Type: `function`

## Syntax

```c
double arctan(double x);
```

## Description

Calculates arctangent of x.

## Return Values

Arcus Tangent of x.

## Example

```c
double x;
x = arctan(0); // result 0
x = arctan(1); // result PI/4
```
