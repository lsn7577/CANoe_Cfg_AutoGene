# getJitterMax

> Category: `Other` | Type: `function`

## Syntax

```c
int getJitterMax()
```

## Description

Determines the upper limit for the allowable deviation when Jitter is set.

## Return Values

Upper deviation in parts per thousand.

## Example

```c
int val;
...
// Assign to val the upper value of the Jitter
val = getJitterMax();
...
```
