# getDrift

> Category: `Other` | Type: `function`

## Syntax

```c
int getDrift()
```

## Description

Determines the constant deviation when Drift is set.

## Return Values

Drift in parts per thousand.

## Example

```c
int val;
...
// Assign to val the Drift value
val = getDrift();
...
```
