# getJitterMin

> Category: `Other` | Type: `function`

## Syntax

```c
int getJitterMin()
```

## Description

Determines the lower limit for the allowable deviation when Jitter is set.

## Return Values

Lower deviation in parts per thousand.

## Example

```c
int val;
...
// Assign to val the lower value of the Jitter
val = getJitterMin();
...
```
