# _Log10

> Category: `Other` | Type: `function`

## Syntax

```c
double _log10(double x)
```

## Description

Calculates the logarithm to base 10.

## Return Values

Logarithm of x (to base 10).

## Example

```c
double x;
x = _log10(100.0); // x == 2.0
```
