# MakeRGB

> Category: `Other` | Type: `function`

## Syntax

```c
long MakeRGB(long Red, long Green, long Blue);
```

## Description

Calculates the color value from the three primary color components.

## Return Values

Color value (type: long)

## Example

```c
MakeRGB(0, 255, 149);
```
