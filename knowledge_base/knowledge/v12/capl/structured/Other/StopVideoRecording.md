# StopVideoRecording

> Category: `Other` | Type: `function`

## Syntax

```c
StopVideoRecording(char windowName[]);
```

## Description

Stops/suspends the recording for a CANoe Video Window.

## Return Values

—

## Example

```c
// stop/suspend the recording for "VideoWindow"
StopVideoRecording("VideoWindow");
```
