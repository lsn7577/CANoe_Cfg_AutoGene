# StartVideoRecording

> Category: `Other` | Type: `function`

## Syntax

```c
StartVideoRecording(char windowName[]);
```

## Description

Starts/resumes the recording for a CANoe Video Window.

## Return Values

—

## Example

```c
// start/resume the recording for "VideoWindow"
StartVideoRecording("VideoWindow");
```
