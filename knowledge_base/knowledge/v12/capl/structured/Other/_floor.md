# _floor

> Category: `Other` | Type: `function`

## Syntax

```c
float _floor(float x);
```

## Description

Calculates the floor of a value, i.e. the largest integer smaller or equal to the value.

## Return Values

Floor of x.

## Example

```c
float x;
x = _floor(3.6); // x == 3.0
```
