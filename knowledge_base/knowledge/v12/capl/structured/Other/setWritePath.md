# setWritePath

> Category: `Other` | Type: `function`

## Syntax

```c
void setWritePath (char relativeOrAbsolutePath[]);
```

## Description

This function sets the write path for the functions openFileWritewriteProfileString andwriteProfileInt ,writeProfileFloat ,. The path can be given as absolute or relative to the currently active configuration.
