# exp

> Category: `Other` | Type: `function`

## Syntax

```c
double exp(double x);
```

## Description

Calculates the exponential function.

## Return Values

Exponent to base e.

## Example

```c
double x;
x = exp(5);
write("Exponent of 5: %f",x); // Result: 148.413159
```
