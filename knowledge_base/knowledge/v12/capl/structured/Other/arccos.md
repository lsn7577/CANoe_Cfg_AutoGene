# arccos

> Category: `Other` | Type: `function`

## Syntax

```c
double arccos(double x);
```

## Description

Calculates arccosine of x.

## Return Values

Arcus Cosine of x.

## Example

```c
double x;
x = arccos(0); // result PI/2
x = arccos(1); // result 0
```
