# ILFaultInjectionResetMsgCycleTime

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILFaultInjectionResetMsgCycleTime(dbMessage msg)
```

## Description

Resets the cycle time of the message to the database cycle time, after having manipulated the cycle time with ILFaultInjectionSetMsgCycleTime.

## Return Values

0: No error.
