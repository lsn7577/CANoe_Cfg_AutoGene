# ILSetSignalRawField

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILSetSignalRawField (dbSignal sig, const char *pData, long aDataLen)
```

## Description

Sets the transferred signal to the provided value.

Use for integer signals > 32 bit.

## Return Values

0: No error.
