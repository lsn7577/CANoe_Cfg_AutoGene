# ILFaultInjectionDisableMsg

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILFaultInjectionDisableMsg (dbMessage msg)
```

## Description

Disables the sending of the message except by calling the function ILSetMsgEvent.

## Return Values

0: No error.
