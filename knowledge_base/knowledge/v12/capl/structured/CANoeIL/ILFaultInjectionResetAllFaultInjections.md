# ILFaultInjectionResetAllFaultInjections

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILFaultInjectionResetAllFaultInjections()
```

## Description

Resets the fault injection settings for all messages of the node.

## Return Values
