# ILFaultInjectionSetMsgDlc

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILFaultInjectionSetMsgDlc (dbMessage msg, dword dlc)
```

## Description

Assigns a new DLC to the message. To set the DLC back to its original value, use ILFaultInjectionResetMsgDlc.

## Return Values

0: No error.
