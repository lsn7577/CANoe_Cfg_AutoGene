# ILFaultInjectionResetMsgLength

> Category: `CANoeIL` | Type: `function`

## Syntax

```c
long ILFaultInjectionResetMsgLength (dbMessage msg)
```

## Description

Resets the message length (in bytes) of the message to the database, after having manipulated the message legth with ILFaultInjectionSetMsgLength.

## Return Values

0: No error.
