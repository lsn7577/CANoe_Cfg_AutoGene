# AvbSetAafChannelCount

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbSetAafChannelCount(dword talkerHandle, dword channelCount);
```

## Description

The function sets the number of channels to be contained in an AAF frame in the stream originated by the Talker.

## Return Values

0: The function completed successfully.
