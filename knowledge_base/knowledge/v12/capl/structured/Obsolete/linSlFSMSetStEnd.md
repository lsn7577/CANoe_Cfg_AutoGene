# linSlFSMSetStEnd

> Category: `Obsolete` | Type: `function`

## Syntax

```c
void linSlFSMSetStEnd(unsigned int slaveId, unsigned int stateId);
```

## Description

Ends the configuration of the state specified by stateId of the Slave with the identifier slaveId.

The entire configuration of a state must appear within a linSlFSMSetStBegin / linSlFSMSetStEnd pair.
