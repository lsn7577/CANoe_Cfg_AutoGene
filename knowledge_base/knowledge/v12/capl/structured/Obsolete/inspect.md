# inspect

> Category: `Obsolete` | Type: `function`

## Syntax

```c
void inspect();
```

## Description

This function causes an update of the variables displayed on the Inspect side of the Write window.

## Return Values

—

## Example

```c
on key 'i'
{
inspect();
// Update of the variables displayed on the Inspect side of the Write window
}
```
