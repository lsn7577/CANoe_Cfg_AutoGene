# LocalSecurityGenerateSHA512

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long LocalSecurityGenerateSHA512 (BYTE[] data, dword dataLength, BYTE[] hash, dword hashLength)
```

## Description

Generates a SHA512 hash for the given data and key. The generated Hash is a SipHash.
