# canGetStdRemoteRate

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long canGetStdRemoteRate (long channel)
```

## Description

Returns the current rate of standard remote CAN frames of the specified channel.

## Parameters

| Name | Description |
|---|---|
| Vector API driver | Values 1 ... 32 |
| Softing API driver | Values 1, 2 |

## Return Values

Current rate of standard remote CAN frames of the specified channel in messages per second.
