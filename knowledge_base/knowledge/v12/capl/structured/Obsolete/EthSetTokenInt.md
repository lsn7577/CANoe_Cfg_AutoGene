# EthSetTokenInt

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long EthSetTokenInt( long packet, char protocolDesignatorl[], char tokenDesignator[], long value ); // form 1
```

## Description

The function sets the integer value of a token.

Form 2 with byte offset sets a part of the token data as integer.

## Return Values

0 or error code
