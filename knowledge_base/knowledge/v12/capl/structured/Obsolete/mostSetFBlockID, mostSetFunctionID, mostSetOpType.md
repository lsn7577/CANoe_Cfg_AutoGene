# mostSetFBlockID, mostSetFunctionID, mostSetOpType

> Category: `Obsolete` | Type: `function`

## Syntax

```c
void mostSetFBlockID(mostMessage msg, long value)
```

## Description

Set the FBlockID, FunctionID or OpType of the message.

Since the FBlockID, FunctionID and OpType are part of the message ID these functions change the ID as well.
