# linSlFSMSetStBegin

> Category: `Obsolete` | Type: `function`

## Syntax

```c
void linSlFSMSetStBegin(unsigned int slaveId, unsigned int stateId);
```

## Description

Prepares the configuration of the state specified by stateId of the Slave with the identifier slaveId.

The entire configuration of a state must appear within one linSlFSMSetStBegin / linSlFSMSetStEnd pair.
