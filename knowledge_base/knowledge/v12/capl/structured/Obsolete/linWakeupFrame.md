> Category: `Obsolete` | Type: `function`

## Syntax

```c
void linWakeupFrame(long syncTimeStamp, long origTimeStamp, byte signal, int external)
```

## Description

Is called whenever a Wakeup frame is recognized. The time stamps are the same as by LINRcvFrame.
