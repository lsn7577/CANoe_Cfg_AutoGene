> Category: `Obsolete` | Type: `function`

## Syntax

```c
long MCDParamResponse ( char moduleName[], char parameterName[], double value)
```

## Description

Callback handler on a MCDGetECUParam request. This function must be defined in the CAPL program to get a response on a read request.
