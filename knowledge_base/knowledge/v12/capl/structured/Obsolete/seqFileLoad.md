# seqFileLoad

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long seqFileLoad(char fileName);
```

## Description

The function opens the file whose name is the string pointed to by name. The file will be opened as read-only.

## Return Values

The return value is the file handle. The return value is <= 0 if an error occurs.
