# AvbGetAafBitDepth

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbGetAafBitDepth(dword listenerOrTalkerHandle, dword& retBitDepth);
```

## Description

The function retrieves the bit depth of an AAF sample in the stream received by the Listener or originated by the Talker.

## Return Values

0: The function completed successfully.
