# AvbGetAafChannelCount

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbGetAafChannelCount(dword listenerOrTalkerHandle, dword& retChannelCount);
```

## Description

The function retrieves the number of channels contained in an AAF frame in the stream received by the Listener or originated by the Talker.

## Return Values

0: The function completed successfully.
