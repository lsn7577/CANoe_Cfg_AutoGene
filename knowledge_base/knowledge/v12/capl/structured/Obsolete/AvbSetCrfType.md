# AvbSetCrfType

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbSetCrfType(dword talkerHandle, dword type);
```

## Description

The function sets the type of time stamp contained in the CRF stream originated by the Talker.

## Return Values

0: The function completed successfully.
