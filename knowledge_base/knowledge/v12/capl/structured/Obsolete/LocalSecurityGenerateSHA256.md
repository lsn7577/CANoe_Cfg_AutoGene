# LocalSecurityGenerateSHA256

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long LocalSecurityGenerateSHA256 (BYTE[] data, dword dataLength, BYTE[] hash, dword hashLength)
```

## Description

Generates a SHA256 hash for the given data and key. The generated Hash is a SipHash.
