# seqFileRewind

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long seqFileRewind(long file);
```

## Description

The function sets the file position indicator to the beginning of the file specified by handle file.

## Return Values

The function returns zero on success. If the operation fails, it returns an non-zero error code.
