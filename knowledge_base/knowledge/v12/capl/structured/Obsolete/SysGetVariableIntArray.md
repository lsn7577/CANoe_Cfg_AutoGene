# SysGetVariableIntArray

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long SysGetVariableIntArray(char namespace[], char variable[], int values[], long arraySize); // form 1
```

## Description

Returns the value of a variable of the int[] type.

## Return Values

0: no error, function successful
