# AvbGetAafFormat

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbGetAafFormat(dword listenerOrTalkerHandle, dword& retFormat);
```

## Description

The function retrieves the format of the AAF samples in the stream received by the Listener or originated by the Talker.

## Return Values

0: The function completed successfully.
