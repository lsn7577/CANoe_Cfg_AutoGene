# AvbGetCrfType

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbGetCrfType(dword listenerOrTalkerHandle, dword& retType);
```

## Description

The function retrieves the type of time stamp contained in the CRF stream received by the Listener or originated by the Talker.

## Return Values

0: The function completed successfully.
