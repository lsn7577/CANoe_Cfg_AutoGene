# linResetScopeTrigger

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long linResetScopeTrigger ()
```

## Description

Resets the oscilloscope trigger condition(s) defined with linSetScopeTrigger.

If this function is called from on-preStart part of the CAPL-program, then this function will have no effect.

## Return Values

On success, a value unequal to zero, otherwise zero.
