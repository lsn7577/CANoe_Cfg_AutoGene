# diagGetNegCode

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long diagGetNegCode (char mnemonic[])
```

## Description

Returns the error code for a KWP2000 mnemonic, e.g. 0x10 for "GR" ("General Reject").

## Return Values

Diagnostics/KWP2000 error code
