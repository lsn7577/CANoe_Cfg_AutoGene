# AvbSetAafFormat

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbSetAafFormat(dword talkerHandle, dword format);
```

## Description

The function sets the format of the AAF samples in the stream originated by the Talker.

## Return Values

0: The function completed successfully.
