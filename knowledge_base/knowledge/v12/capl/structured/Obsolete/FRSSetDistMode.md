# FRSSetDistMode

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long FRSSetDistMode (int triggerCondition, int mode);
```

## Description

Sets the disturbance type. A bitstream disturbance generates the disturbance sequence after the trigger is detected. The frame disturbance changes individual bits within a frame with bit accuracy.

## Return Values

0: successful
