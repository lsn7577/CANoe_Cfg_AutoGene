# AvbGetAafSampleRate

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbGetAafSampleRate(dword listenerOrTalkerHandle, dword& retSampleRate);
```

## Description

The function retrieves the sample rate of the AAF stream received by the Listener or originated by the Talker.

## Return Values

0: The function completed successfully.
