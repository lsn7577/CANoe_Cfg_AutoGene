# AvbGetCrfAudioSampleTimestampInterval

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbGetCrfAudioSampleTimestampInterval(dword listenerOrTalkerHandle, dword& retTimestampInterval);
```

## Description

The function retrieves the interval between each Audio Sample Timestamp in the CRF stream received by the Listener or originated by the Talker.

## Return Values

0: The function completed successfully.
