# AvbSetAafSampleRate

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbSetAafSampleRate(dword talkerHandle, dword sampleRate);
```

## Description

The function sets the sample rate of the AAF stream originated by the Talker.

## Return Values
