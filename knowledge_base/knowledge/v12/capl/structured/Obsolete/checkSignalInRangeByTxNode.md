# checkSignalInRangeByTxNode

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long checkSignalInRangeByTxNode (dbSignal aSignal, dbNode aTxNode, float aLowLimit, float aHighLimit);
```

## Description

Checks the signal value against the condition:

aLowLimit <= Value <= aHighLimit

## Return Values

1: If the condition is TRUE
