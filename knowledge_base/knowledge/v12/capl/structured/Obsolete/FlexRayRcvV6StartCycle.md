# FlexRayRcvV6StartCycle

> Category: `Obsolete` | Type: `function`

## Syntax

```c
FlexRayRcvV6StartCycle(long msgTime, byte nmlen, byte nmdata[])
```

## Description

A function defined in CAPL with this signature receives all FlexRay StartCycle events.The event is generated at the beginning of a communication cycle.It contains the NM vector that is valid for this round.
