# AvbSetAafBitDepth

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbSetAafBitDepth(dword talkerHandle, dword bitDepth);
```

## Description

The function sets the bit depth of an AAF sample in the stream originated by the Talker.

## Return Values

0: The function completed successfully.
