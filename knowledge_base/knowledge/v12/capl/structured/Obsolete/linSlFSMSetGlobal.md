# linSlFSMSetGlobal

> Category: `Obsolete` | Type: `function`

## Syntax

```c
void linSlFSMSetGlobal(unsigned int slaveId, unsigned int numberOfStates);
```

## Description

Sets the number of states given in numberOfStates for the Slave with identifier slaveId.

This function must be called before the Slave's states are configured.
