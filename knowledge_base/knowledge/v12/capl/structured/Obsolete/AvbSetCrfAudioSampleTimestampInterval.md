# AvbSetCrfAudioSampleTimestampInterval

> Category: `Obsolete` | Type: `function`

## Syntax

```c
dword AvbSetCrfAudioSampleTimestampInterval(dword talkerHandle, dword timestampInterval);
```

## Description

The function sets the interval between each Audio Sample Timestamp in the CRF stream originated by the Talker.

## Return Values

0: The function completed successfully.
