# MCDMapECUParamToSysVariableWrite

> Category: `Obsolete` | Type: `function`

## Syntax

```c
long MCDMapECUParamToSysVariableWrite(char moduleName[], char parameterName[], SysVarName)
```

## Description

Maps an ECU parameter to a system variable so that it is changed whenever the system variable changes.

## Return Values

0: No error.
