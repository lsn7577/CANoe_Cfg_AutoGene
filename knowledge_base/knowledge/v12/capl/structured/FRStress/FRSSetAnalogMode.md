# FRSSetAnalogMode

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSSetAnalogMode (float baudrate, int channel, int payloadLength)
```

## Description

Activates the analog disturbance mode of FRstress.Configuration functions that belong to another operating mode are ignored.

## Return Values

0: successful
