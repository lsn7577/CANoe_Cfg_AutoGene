# FRSEnableTrigDist

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSEnableTrigDist(int triggerCondition, int iEnable)
```

## Description

Activates a trigger/disturbance page of FRstress. This function can be called out of a measurement session.

## Return Values

0: successful
