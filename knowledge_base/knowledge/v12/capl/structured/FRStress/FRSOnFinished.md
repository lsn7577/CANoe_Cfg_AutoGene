# FRSOnFinished

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSOnFinished(char CallbackName[])
```

## Description

Registers a callback. The callback will be executed when the dedicated disturbance counter is 0. The callback function must be added in the CAPL section callback.

## Return Values

0: successful
