# FRSSetScopeMode

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSSetScopeMode( float baudrate, int channel, int payloadLength);
```

## Description

Activates the Scope mode of FRstress.Configuration functions that belong to another operating mode are ignored.

## Return Values

0: successful
