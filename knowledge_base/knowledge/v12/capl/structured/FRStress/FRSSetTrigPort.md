# FRSSetTrigPort

> Category: `FRStress` | Type: `function`

## Syntax

```c
long FRSSetTrigPort (int triggerCondition, int portMask);
```

## Description

Adds a bit sequence to the the respective trigger condition as a trigger value. A complex trigger condition can be created through multiple calls.

## Return Values

0: successful
