# sysDefineVariableFloatArray

> Category: `SystemVariables` | Type: `function`

## Syntax

```c
long sysDefineVariableFloatArray(char namespace[], char vari-able[], float initialValues[], long arraySize); // form 1
```

## Description

Defines a variable of the float[] type.

## Return Values

0: no error, function successful
