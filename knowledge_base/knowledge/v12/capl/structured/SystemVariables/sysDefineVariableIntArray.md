# sysDefineVariableIntArray

> Category: `SystemVariables` | Type: `function`

## Syntax

```c
long sysDefineVariableIntArray(char namespace[], char variable[], int initialValues[], long arraySize); // form 1
```

## Description

Defines a variable of the int[] type.

## Return Values

0: no error, function successful
