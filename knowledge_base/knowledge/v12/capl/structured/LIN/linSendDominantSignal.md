# linSendDominantSignal

> Category: `LIN` | Type: `function`

## Syntax

```c
dword linSendDominantSignal(dword lengthInMicroseconds);
```

## Description

Sends a dominant signal of the specified length.

## Return Values

On success, a value unequal to zero, otherwise zero.
