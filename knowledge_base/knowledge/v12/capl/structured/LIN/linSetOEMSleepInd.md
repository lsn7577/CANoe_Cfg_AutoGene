# linSetOEMSleepInd

> Category: `LIN` | Type: `function`

## Syntax

```c
long linSetOEMSleepInd(long active)
```

## Description

Sets/resets the sleep indication bit for a calling slave node.

## Return Values

On success, a value unequal to zero, otherwise zero.
