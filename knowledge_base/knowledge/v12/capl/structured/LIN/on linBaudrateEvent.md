# on linBaudrateEvent

> Category: `LIN` | Type: `event`

## Description

The event procedure on linBaudrateEvent is called when the LIN hardware has successfully measured the baud rate of an external master or if the baud rate deviates by more than 0.5% from the last reported value. This only occurs if an external master is being used (in which case the LIN hardware is running in Slave mode).

The keyword this and the selectors can be used to access the data of the event that has just been received.
