# linSetRespTolerance

> Category: `LIN` | Type: `function`

## Syntax

```c
dword linSetRespTolerance(long frameId, long toleranceInPercent)
```

## Description

Sets the response tolerance for a specified frame in percent.

## Return Values

On success, a value unequal to zero, otherwise zero.
