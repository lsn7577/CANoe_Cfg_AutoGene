# linSetOEMWakeupInd

> Category: `LIN` | Type: `function`

## Syntax

```c
long linSetOEMWakeupInd(long active)
```

## Description

Sets/resets the wake-up indication bit for a calling slave node.

## Return Values

On success, a value unequal to zero, otherwise zero.
