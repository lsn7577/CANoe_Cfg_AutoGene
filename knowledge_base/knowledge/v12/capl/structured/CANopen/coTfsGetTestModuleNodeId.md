# coTfsGetTestModuleNodeId

> Category: `CANopen` | Type: `function`

## Syntax

```c
dword coTfsGetTestModuleNodeId( void );
```

## Description

The function gets the node-ID of the tester.

## Return Values

Node-ID of the tester.

## Example

```c
dword nodeId;
nodeId = coTfsGetTestModuleNodeId();
```
