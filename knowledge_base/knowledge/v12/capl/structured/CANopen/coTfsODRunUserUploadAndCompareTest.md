# coTfsODRunUserUploadAndCompareTest

> Category: `CANopen` | Type: `function`

## Syntax

```c
long coTfsODRunUserUploadAndCompareTest( void );
```

## Description

This function executes a SDO Upload for all objects that are defined in the internal list of test objects (see coTfsODAddEntry) and checks if the read data are conform to the stored data.

## Return Values

Error code

see example of coTfsODAddEntrySubIndexRange
