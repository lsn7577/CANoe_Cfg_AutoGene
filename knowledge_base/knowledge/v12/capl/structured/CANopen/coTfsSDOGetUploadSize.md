# coTfsSDOGetUploadSize

> Category: `CANopen` | Type: `function`

## Syntax

```c
dword coTfsSDOGetUploadSize( void );
```

## Description

Provide the size of the transferred data after a SDO upload. This can be an expedited, segmented or block transfer.

## Return Values

Transfer size
