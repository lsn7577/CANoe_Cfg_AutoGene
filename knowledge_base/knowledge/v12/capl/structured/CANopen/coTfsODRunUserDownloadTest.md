# coTfsODRunUserDownloadTest

> Category: `CANopen` | Type: `function`

## Syntax

```c
long coTfsODRunUserDownloadTest( void );
```

## Description

This function executes a SDO Download for all objects that are defined in the internal list of test objects (see coTfsODAddEntry).

## Return Values

Error code
