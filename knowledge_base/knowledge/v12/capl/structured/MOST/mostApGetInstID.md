# mostApGetInstID

> Category: `MOST` | Type: `function`

## Syntax

```c
long mostApGetInstID()
```

## Description

Returns the InstID of the CAPL node. The function block must be assigned to the CAPL program exclusively via mostApRegister (fblockID, instIDDefault) or the database.

## Return Values

>0: Valid InstID
