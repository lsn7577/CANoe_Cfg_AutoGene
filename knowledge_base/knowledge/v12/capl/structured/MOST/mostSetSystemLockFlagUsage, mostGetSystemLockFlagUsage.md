# mostSetSystemLockFlagUsage, mostGetSystemLockFlagUsage

> Category: `MOST` | Type: `function`

## Syntax

```c
long mostSetSystemLockFlagUsage(long channel, long mode)
```

## Description

For test purposes the stress network interface controller in the OptoLyzer G2 3150o is able to suppress the system lock flag for the next node by calling.

mostSetSystemLockFlagUsage with mode 0.

## Return Values

mostSetSystemLockFlagUsage: See error codes
