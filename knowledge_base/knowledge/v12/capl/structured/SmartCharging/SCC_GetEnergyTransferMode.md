# SCC_GetEnergyTransferMode

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_GetEnergyTransferMode ( long Index, char EnergyTransferMode[] )
```

## Description

Gets the encrypted private key of the new contract certificate.

## Return Values

The EnergyTransferMode within the specified list index (to the output buffer).
