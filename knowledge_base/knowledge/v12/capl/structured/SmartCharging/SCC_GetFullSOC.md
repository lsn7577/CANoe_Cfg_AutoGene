# SCC_GetFullSOC

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_GetFullSOC ( )
```

## Description

Gets the FullSOC value of a ChargeParameterDiscoveryReq message.

## Return Values

Charging status in which the vehicle regards the battery as fully charged, in percent.
