# SCC_GetPaymentOptions

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_GetPaymentOptions ( )
```

## Description

Gets the Payment options where 0=ExternalPayment, 1=Contract and 2=both

## Return Values

Available PaymentOptions, where
0 = ExternalPayment
1 = Contract
2 = both
