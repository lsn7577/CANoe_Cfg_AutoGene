# SCC_GetRetryCounter

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_Get Retry Counter ()
```

## Description

Gets the retry counter of a CertificationUpdateRes.

In case of failure, this denotes when the EVCC should try to get the new Certificate again (number of days).
