# SCC_CreateSECCDiscoveryRes

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_CreateSECCDiscoveryRes ( char IpAddress[],dword Port, dword Security )
```

## Description

Creates a SECC Discovery Response message for sending. Depending on the function call, the IP Address is either interpreted as an IPv4 address or an IPv6 address.

## Return Values

1 if successful
