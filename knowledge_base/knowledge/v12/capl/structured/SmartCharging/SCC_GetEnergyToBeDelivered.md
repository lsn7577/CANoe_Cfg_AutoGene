# SCC_GetEnergyToBeDelivered

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetEnergyToBeDelivered ( )
```

## Description

Gets EnergyTransferMode with the target index (ISO 15118).

## Return Values

Energy to be delivered by the charge point.
