# SCC_GetServiceParameterSetData

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetServiceParameterSetData ( long Index, long& ParameterSetID, long& ParameterCount )
```

## Description

Gets the ParameterSetID with the specified list index < ParameterSetCount.

## Return Values

ParameterSetID and number of parameters of the parameter set with the specified list index < ParameterSetCount (via reference).
