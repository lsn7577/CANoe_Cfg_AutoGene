# SCC_SupportedAppProtocolReq

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_SupportedAppProtocolReq
```

## Description

The callback is called as soon as a SupportedAppProtocol Request is received.

The list entries must be queried via the separate help function SCC_GetAppProtocolData.
