# SCC_GetOEMPRovisioningCertificate

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetOEMPRovisioningCertificate ( char Certificate[] )
```

## Description

Reads the OEM provisioning certificate from the target certificate as base64 string.

## Return Values

Reads the OEM provisioning certificate from the target certificate as base64 string (to the output buffer).
