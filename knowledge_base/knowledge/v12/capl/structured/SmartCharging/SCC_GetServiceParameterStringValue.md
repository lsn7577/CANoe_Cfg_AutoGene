# SCC_GetServiceParameterStringValue

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetServiceParameterStringValue ( long i1, long i2, char Value[] )
```

## Description

Gets the string value of the parameter with the selected indices.

## Return Values

The string value of the parameter with the selected indices, to the output buffer. Use for value type stringValue.
