# SCC_GetEVSENotification

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetEVSENotification ( char EVSENotification, long& NotificationMaxDelay )
```

## Description

Returns the EVSENotification and its maximum delay, if present, from the EVSE status.

## Return Values

EVSENotification and its maximum delay, if present, from DC_EVSEStatus
