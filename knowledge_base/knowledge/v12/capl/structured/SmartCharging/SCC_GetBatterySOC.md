# SCC_GetBatterySOC

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetBatterySOC ( )
```

## Description

Outputs the current charging state of the battery, either in [Wh] with the first function or in [%] with the second

## Return Values

Charging state in Wh.
