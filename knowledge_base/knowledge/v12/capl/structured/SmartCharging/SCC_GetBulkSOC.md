# SCC_GetBulkSOC

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_GetBulkSOC( )
```

## Description

Gets the BulkSOC value of a ChargeParameterDiscoveryReq message.

## Return Values

Charging status in which the vehicle regards a bulk charging process as finished, in percent.
