# SCC_SetMinCurrent

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_SetMinCurrent ( float MinCurrent )
```

## Description

Sets the minimum current for the message ChargeParameterDiscoveryReq. A default value can be set using the configuration file.

## Return Values

0: Not successful
