# SCC_GetPeakCurrentRipple

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetPeakCurrentRipple ( )
```

## Description

Get the value of PeakCurrentRipple of a ChargeParameterDiscoveryRes message.

## Return Values

Peak-to-peak magnitude of the current ripple.
