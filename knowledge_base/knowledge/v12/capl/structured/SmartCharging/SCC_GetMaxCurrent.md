# SCC_GetMaxCurrent

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetMaxCurrent ( )
```

## Description

Returns the SDP or V2G message header (to the output buffer).

## Return Values

Returns Maximum line current of vehicle (AC: EVMaxCurrent, DC: EVMaximumCurrentLimit).
