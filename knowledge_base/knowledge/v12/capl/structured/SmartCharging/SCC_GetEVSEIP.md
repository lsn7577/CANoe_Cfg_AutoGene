# SCC_GetEVSEIP

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetEVSEIP ( byte IpAddress[] )
```

## Description

Gets the EVSE IP address of a SECCDiscoveryRes message.

## Return Values

IP (v6) transmitted from the charge point (16 byte, to the output buffer)
