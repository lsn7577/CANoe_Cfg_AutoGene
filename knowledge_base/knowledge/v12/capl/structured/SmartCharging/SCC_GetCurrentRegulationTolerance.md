# SCC_GetCurrentRegulationTolerance

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetCurrentRegulationTolerance ( )
```

## Description

Get current regulation tolerance of a ChargeParameterDiscoveryRes message.

## Return Values
