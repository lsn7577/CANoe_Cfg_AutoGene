# SCC_GetSelectedParameterSetID

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_ GetSelectedParameterSetID ( long Index )
```

## Description

Gets the parameter set ID of a ServicePaymentSelectionReq message.

## Return Values

ID of the selected service’s parameter set with the specified list index < ServiceCount.
