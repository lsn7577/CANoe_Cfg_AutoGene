# SCC_SLAC_SetLinkStatus

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_SLAC_SetLinkStatus ( long LinkStatus )
```

## Description

This function overrides the internally stored link status of the QCA7000/7005 chip. This is useful if no hardware is present.

## Return Values

0: Not successful
