# SCC_GetDepartureTime

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
dword SCC_GetDepartureTime( )
```

## Description

Gets the end of charging time ChargeParameterDiscoveryReq message.

## Return Values

End of charging time as offset in seconds from the time this message is sent.
