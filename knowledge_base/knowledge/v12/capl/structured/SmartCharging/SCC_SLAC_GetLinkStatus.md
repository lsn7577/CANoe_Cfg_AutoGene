# SCC_SLAC_GetLinkStatus

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_SLAC_GetLinkStatus ( )
```

## Description

This function retrieves the internally stored link status of the QCA7000/7005 chip.

## Return Values

0: No link available
