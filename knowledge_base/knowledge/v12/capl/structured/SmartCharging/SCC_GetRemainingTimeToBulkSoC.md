# SCC_GetRemainingTimeToBulkSoC

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetRemainingTimeToBulkSoC ( )
```

## Description

Gets the RemainingTimeToBulkSoC of a CurrentDemandReq message.

## Return Values

Remaining time to end of a bulk charging operation in seconds.
