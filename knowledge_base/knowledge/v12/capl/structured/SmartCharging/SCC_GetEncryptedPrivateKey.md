# SCC_GetEncryptedPrivateKey

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
void SCC_GetEncryptedPrivateKey ( byte Key[], char IdAttr[] )
```

## Description

Gets the encrypted private key of the new contract certificate.

## Return Values

Gets the encrypted private key (48 byte) of the new contract certificate and its Id attribute (to the output buffers).
