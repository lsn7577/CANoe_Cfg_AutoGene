# SCC_GetMaxVoltage

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetMaxVoltage ( )
```

## Description

Gets the vehicle's maximum voltage.

## Return Values

Returns Maximum line voltage of vehicle (AC: EVSEMaxVoltage, DC: EVSEMaximumVoltageLimit).
