# SCC_GetNominalVoltage

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
float SCC_GetNominalVoltage ( )
```

## Description

Gets the charge point's nominal voltage from a ChargeParameterDiscoveryRes message.

## Return Values

Supported line voltage of charge point.
