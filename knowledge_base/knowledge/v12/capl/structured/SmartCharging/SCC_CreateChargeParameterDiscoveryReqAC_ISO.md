# SCC_CreateChargeParameterDiscoveryReqAC_ISO

> Category: `SmartCharging` | Type: `function`

## Syntax

```c
long SCC_CreateChargeParameterDiscoveryReqAC_ISO ( byte SessionID[], char EnergyTransferType[], float EAmount, float MaxVoltage, float MaxCurrent, float MinCurrent )
```
