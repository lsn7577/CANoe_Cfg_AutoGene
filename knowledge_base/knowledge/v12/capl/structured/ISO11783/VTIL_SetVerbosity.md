# VTIL_SetVerbosity

> Category: `ISO11783` | Type: `function`

## Syntax

```c
dword VTIL_SetVerbosity( dword verbosityLevel); // form 1
```

## Description

Sets verbosity for writing in Write Window.

Form 2 applicable in test module / test unit only.

## Return Values

0: Function has been executed successfully
