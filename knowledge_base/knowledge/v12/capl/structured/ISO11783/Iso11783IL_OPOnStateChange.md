# Iso11783IL_OPOnStateChange

> Category: `ISO11783` | Type: `function`

## Syntax

```c
void Iso11783IL_OPOnStateChange( dword state );
```

## Description

The function is called by the node layer, if the state of the Object Pool API has changed.
