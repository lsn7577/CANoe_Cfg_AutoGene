# Iso11783IL_OPESC

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long Iso11783IL_OPESC( ); // form 1
```

## Description

The function aborts user input on the Virtual Terminal. A ESC command is sent to the Virtual Terminal.

## Example

```c
Iso11783IL_OPESC( );
```
