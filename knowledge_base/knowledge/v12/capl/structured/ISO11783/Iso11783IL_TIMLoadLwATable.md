# Iso11783IL_TIMLoadLwATable

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long Iso11783IL_TIMLoadLwATable(char iniFilePath[]); // form 1
```

## Description

Loads a table with lightweight authentication (LwA) keys from a file.

The LwA table has to be stored with function Iso11783IL_TIMSaveLwATable before.

## Return Values

0: Property has been set successfully
