# Iso11783GetLastError

> Category: `ISO11783` | Type: `function`

## Syntax

```c
dword Iso11783GetLastError( void );
```

## Description

This function returns the last error code.

## Return Values

Error code

## Example

```c
dword errCode;
errCode = Iso11783GetLastError();
```
