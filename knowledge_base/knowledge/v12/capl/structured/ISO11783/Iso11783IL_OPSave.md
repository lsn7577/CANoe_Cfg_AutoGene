# Iso11783IL_OPSave

> Category: `ISO11783` | Type: `function`

## Syntax

```c
long Iso11783IL_OPSave( char filename[] ); // form 1
```

## Description

The function saves an object pool file (*.iop).

## Return Values

0: Function has been executed successfully

## Example

```c
Iso11783IL_OPSave( 
 "ObjectPool.iop");
```
